2K近邻法 (KNN) 分类与预测

来源：lec3KNN.pdf · class包实现KNN分类，kknn包实现加权KNN
KNN核心函数语法class
knn(train=训练样本集, test=测试样本集, cl=输出变量, k=近邻个数K)
knn1(train=训练样本集, test=测试样本集, cl=输出变量)        # K=1的特例
knn.cv(train=训练样本集, cl=输出变量, k=近邻个数)           # 留一交叉验证
说明：
  • train 训练集特征矩阵，test 测试集特征矩阵，cl 训练集的类别标签
  • knn.cv() 自动执行留一交叉验证（Leave-One-Out CV），无需手动划分测试集
  • prob=TRUE 可返回各观测属于各类别的比例
KNN 模拟数据生成class
set.seed(12345)
x1 <- runif(60, -1, 1)                          # 生成60个[-1,1]均匀分布数
x2 <- runif(60, -1, 1)
y <- sample(c(0,1), size=60, replace=TRUE, prob=c(0.3, 0.7))  # 0.7:0.3随机指派类别
Data <- data.frame(Fx1=x1, Fx2=x2, Fy=y)
SampleId <- sample(x=1:60, size=18)             # 随机抽取18个样本
DataTest <- Data[SampleId,]                     # 测试集
DataTrain <- Data[-SampleId,]                   # 训练集（取反）
说明：
  • set.seed() 固定随机种子，确保结果可重复
  • runif(n, min, max) 生成均匀分布随机数
  • sample() 随机抽样，prob参数指定各类比例
  • Data[-SampleId,] 用负索引排除测试集样本，得到训练集
KNN 三种评估方法：旁置法错判率class
library("class")
errRatio1 <- vector()                     # 初始化错判率向量
for(i in 1:30){                           # K从1到30
  KnnFit <- knn(train=DataTrain[,1:2], test=DataTest[,1:2],
                cl=DataTrain[,3], k=i)    # 训练集≠测试集
  CT <- table(DataTest[,3], KnnFit)       # 混淆矩阵
  errRatio1 <- c(errRatio1, (1-sum(diag(CT))/sum(CT))*100)
}
plot(errRatio1, type="l", xlab="近邻个数K", ylab="错判率(%)")
说明：
  • 旁置法(Holdout)：训练集和测试集不同，是最简单的评估方式
  • table(真实值, 预测值) 生成混淆矩阵，对角线元素为正确分类数
  • sum(diag(CT))/sum(CT) 计算正确率，(1-正确率)*100 得到错判率(%)
KNN 留一法 (LOOCV) 错判率class
set.seed(12345)
errRatio2 <- vector()
for(i in 1:30){
  KnnFit <- knn.cv(train=Data[,1:2], cl=Data[,3], k=i)  # 留一法
  CT <- table(Data[,3], KnnFit)
  errRatio2 <- c(errRatio2, (1-sum(diag(CT))/sum(CT))*100)
}
lines(1:30, errRatio2, col=2)            # 红色线叠加到图上
说明：
  • 留一法(LOOCV)：knn.cv() 对每个样本用其余 N-1 个样本训练，对该样本预测，循环N次
  • 评估结果更接近真实泛化误差，但计算量最大
加权KNN (kknn包) 与核函数比较kknn

library("kknn")

                   distance=2, 
                   kernel=c("rectangular", "triangular", "gaussian"))
plot(fit$MISCLASS[,1]*100, type="l", main="不同核函数的错判率曲线")
lines(fit$MISCLASS[,2]*100, lty=2, col=1)

说明：
  • train.kknn() 自动训练不同K值和核函数下的模型
  • kernel=可选三种核函数：rectangular(均匀核)、triangular(三角核)、gaussian(高斯核)
  • distance=2 使用欧氏距离（闵可夫斯基距离 p=2）
  • 除均匀核外，其他核函数的预测误差差异不明显

3决策树、袋装、推进、随机森林

来源：lec4-DT.pdf · 这是R代码最丰富的一章，涵盖rpart/ipred/adabag/randomForest
rpart() 建立分类回归树rpart / rpart.plot
install.packages("rpart")
install.packages("rpart.plot")
library("rpart")
library("rpart.plot")

BuyOrNot <- read.table(file="消费决策数据.txt", header=TRUE)
BuyOrNot$Income  <- as.factor(BuyOrNot$Income)    # 转为因子型
BuyOrNot$Gender  <- as.factor(BuyOrNot$Gender)

Ctl <- rpart.control(minsplit=2, maxcompete=4, xval=10, 
                      maxdepth=10, cp=0)            # 设置控制参数
set.seed(12345)
TreeFit1 <- rpart(Purchase~., data=BuyOrNot, method="class",
                  parms=list(split="gini"), control=Ctl)
rpart.plot(TreeFit1, type=4, branch=0, extra=2)     # 可视化决策树
plotcp(TreeFit1)                                      # 可视化CP值
说明：
  • rpart() 是R中建立决策树的核心函数
  • method="class" 建立分类树，"anova"建立回归树
  • split="gini" 用基尼系数选择分裂，"information"用信息熵
  • rpart.control参数：minsplit节点最小样本量/maxdepth最大深度/cp复杂度参数/xval交叉验证折数
  • rpart.plot(type=4, extra=2) 显示每个节点的预测类别和比例
剪枝：prune() 最小代价复杂度剪枝rpart
TreeFit3 <- prune(TreeFit1, cp=0.008)      # 按指定CP值剪枝
rpart.plot(TreeFit3, type=4, branch=0, extra=2)
printcp(TreeFit3)                            # 浏览CP值与误差关系
# CP值越大，剪掉的枝越多，树越简单
说明：
  • prune(tree, cp=0.008) 剪掉复杂度参数大于0.008的子树
  • CP(Complexity Parameter)控制代价复杂度：Rα(T)=R(T)+α|T|，α=CP
  • CP越大，树越简单（叶节点越少），但预测误差可能增大
  • printcp() 输出交叉验证误差xerror，选xerror最小时的CP值
Bagging 袋装技术 (ipred包)ipred
install.packages("ipred")
library("ipred")

Ctl <- rpart.control(minsplit=20, maxcompete=4, maxdepth=30, 
                      cp=0.01, xval=10)
set.seed(12345)
BagM1 <- bagging(MAILSHOT~., data=MailShot, nbagg=25, 
                  coob=TRUE, control=Ctl)
# OOB estimate of misclassification error: 44.67%

CFit2 <- predict(BagM1, MailShot)           # 预测
ConfM2 <- table(MailShot$MAILSHOT, CFit2)
(E2 <- (sum(ConfM2)-sum(diag(ConfM2)))/sum(ConfM2))
# 0.2233333 = 22.3% 误判率
说明：
  • bagging() 实现Bootstrap Aggregating（袋装技术）
  • nbagg=25 Bootstrap重复25次，生成25棵决策树
  • coob=TRUE 基于袋外(Out-Of-Bag)观测计算预测误差（约36.8%样本未被抽中）
  • Bagging通过多个模型的投票(分类)或平均(回归)降低方差
Boosting 推进技术 (adabag包)adabag
install.packages("adabag")
library("adabag")

Ctl <- rpart.control(minsplit=20, maxcompete=4, maxdepth=30, 
                      cp=0.01, xval=10)
set.seed(12345)
BoostM <- boosting(MAILSHOT~., data=MailShot, boos=TRUE,
                    mfinal=25, coeflearn="Breiman", control=Ctl)
BoostM$importance              # 变量重要性得分
ConfM4 <- table(MailShot$MAILSHOT, BoostM$class)
(E4 <- (sum(ConfM4)-sum(diag(ConfM4)))/sum(ConfM4))
# 0.0267 = 2.7% 误判率 (远优于单棵树28.3%和Bagging22.3%)
说明：
  • boosting() 实现AdaBoost算法，属于串行集成
  • mfinal=25 迭代25轮，每轮训练一个弱分类器
  • boos=TRUE 每轮根据上一轮误判情况调整样本权重（错分样本权重增大）
  • 返回值包括trees(各棵树)、weights(各模型投票权重)、importance(变量重要性)

4人工神经网络 (neuralnet / nnet)

来源：lec6-AN.pdf · neuralnet包实现可视化BP网络，nnet包实现传统三层BP网络
neuralnet() 建立BP神经网络neuralnet
install.packages("neuralnet")
library("neuralnet")

BuyOrNot <- read.table(file="消费决策数据.txt", header=TRUE)
BPnet1 <- neuralnet(Purchase~Age+Gender+Income, data=BuyOrNot,
                     hidden=2, err.fct="ce", linear.output=FALSE)
BPnet1$result.matrix        # 查看连接权重和迭代信息
plot(BPnet1)                # 绘制神经网络拓扑图
说明：
  • hidden=2 单个隐藏层，2个隐节点
  • err.fct="ce" 使用交叉熵(Cross-Entropy)作为损失函数（分类问题推荐）
  • linear.output=FALSE 输出层使用非线性激活函数（分类），TRUE用于回归
  • $result.matrix包含各层连接权重，如Age.to.1layhid1、1layhid.1.to.Purchase
  • $weight是列表形式存储的各层权重矩阵
广义权重 gwplot() — 评价输入变量重要性neuralnet
head(BPnet1$generalized.weights[[1]])   # 前6个观测的广义权重
par(mfrow=c(2,2))
gwplot(BPnet1, selected.covariate="Age")
gwplot(BPnet1, selected.covariate="Gender")
gwplot(BPnet1, selected.covariate="Income")
说明：
  • 广义权重(Generalized Weights)：衡量每个输入变量对输出变量预测的边际影响
  • gwplot() 绘制广义权重随输入变量取值的分布图
  • 广义权重的绝对值越大，该输入变量对预测越重要
  • 权重接近0意味着该变量对输出几乎没有影响
compute() 模拟不同输入组合的预测neuralnet
newData <- matrix(c(39,1,1, 39,1,2, 39,1,3,
                     39,2,1, 39,2,2, 39,2,3), 
                   nrow=6, ncol=3, byrow=TRUE)
new.output <- compute(BPnet1, covariate=newData)
new.output$net.result          # 查看不同组合的预测概率
说明：
  • compute() 计算任意输入变量组合下输出节点的预测值
  • 可用于对比分析：如固定年龄=39，改变性别和收入，观察购买概率变化
  • 典型发现：女性购买概率普遍高于男性，高收入人群购买概率更低
ROC曲线确定最优概率分割值ROCR
install.packages("ROCR")
library("ROCR")
summary(BPnet1$net.result[[1]])          # 浏览预测概率分布
pred <- prediction(predictions=as.vector(BPnet1$net.result),
                   labels=BPnet1$response)
par(mfrow=c(2,1))
perf <- performance(pred, measure="tpr", x.measure="fpr")
plot(perf, colorize=TRUE, print.cutoffs.at=c(0.2,0.45,0.46,0.47))
perf <- performance(pred, measure="acc")
plot(perf)    # 概率分割值与预测精度的关系
说明：
  • prediction() 将预测概率和真实标签打包
  • performance(measure="tpr", x.measure="fpr") 计算ROC曲线的真阳率vs假阳率
  • ROC曲线越接近左上角，模型分类性能越好
  • 选TPR大且FPR小的概率值作为分割阈值（不一定是0.5）
nnet() 传统三层BP反向传播网络nnet
install.packages("nnet")
library("nnet")
set.seed(1000)
BPnet2 <- nnet(Purchase~Age+Gender+Income, data=BuyOrNot,
                size=2, entropy=TRUE, abstol=0.01)
predict(BPnet2, BuyOrNot, type="class")   # 输出预测类别（不是概率）
说明：
  • nnet() 建立传统三层BP网络（输入层→单隐藏层→输出层）
  • size=2 隐藏层2个节点
  • entropy=TRUE 使用交叉熵损失函数；FALSE使用误差平方和
  • maxit=100 默认最大迭代次数100
  • neuralnet vs nnet区别：neuralnet支持多隐藏层+可视化，nnet只支持单隐藏层但更轻量

5支持向量机 (SVM)

来源：lec9-Mlearning.pdf · e1071包实现SVM分类，tune.svm()自动超参数调优
svm() 核心函数语法e1071
svm(formula=公式, data=数据框名, scale=TRUE/FALSE, 
    type=支持向量机类型, kernel=核函数名, 
    gamma=g, degree=d, cost=C)
# type: "C-classification"(分类), "eps-regression"(回归)
# kernel: "linear"(线性核), "polynomial"(多项式核), "radial"(径向基核)
# gamma: 径向基核/多项式核的参数  cost: 损失惩罚参数C
说明：
  • scale=TRUE 建模前自动对数据标准化（推荐打开，因为SVM依赖距离度量）
  • cost=C 控制软间隔的惩罚力度：C大→过拟合（强制完美分类），C小→欠拟合（容忍误分）
  • 返回值：$SV支持向量、$index支持向量索引、$decision.values决策函数值
线性可分SVM：比较不同cost的影响e1071
library("e1071")
# cost=10: 惩罚大，要求更精确分类，支持向量少
SvmFit <- svm(Fy~., data=data_train, type="C-classification",
              kernel="linear", cost=10, scale=FALSE)
SvmFit$index        # 16个支持向量
# cost=0.1: 惩罚小，容忍更多误分，支持向量多
SvmFit2 <- svm(Fy~., data=data_train, type="C-classification",
               kernel="linear", cost=0.1, scale=FALSE)
SvmFit2$index       # 25个支持向量（更多！）
plot(x=SvmFit, data=data_train, formula=Fx1~Fx2, 
     svSymbol="#", dataSymbol="*", grid=100)   # 可视化决策边界
说明：
  • cost越大→惩罚越重→间隔越小→支持向量越少→模型越复杂
  • cost越小→惩罚越轻→间隔越大→支持向量越多→模型越简单
  • plot(x=svm对象) 自动绘制决策边界和支持向量（支持向量用#标记）
tune.svm() 自动10折交叉验证调优e1071
set.seed(12345)
tObj <- tune.svm(Fy~., data=data_train, type="C-classification",
                  kernel="linear", 
                  cost=c(0.001, 0.01, 0.1, 1, 5, 10, 100, 1000),
                  scale=FALSE)
summary(tObj)
# best parameters: cost = 1
# best performance: 0.15 (15%误差率)

BestSvm <- tObj$best.model       # 提取最优模型
yPred <- predict(BestSvm, data_test)
(ConfM <- table(yPred, data_test$Fy))
(Err <- (sum(ConfM)-sum(diag(ConfM)))/sum(ConfM))   # 0.1
说明：
  • tune.svm() 对cost和gamma的每个组合做10折交叉验证，找出最优参数
  • $best.parameters最优参数组合、$best.performance最优性能、$best.model最优模型
  • cost参数向量覆盖[0.001, 1000]的宽范围，确保找到全局最优
非线性SVM + 径向基核 调优e1071
set.seed(12345)
tObj <- tune.svm(Fy~., data=data_train, type="C-classification",
                  kernel="radial", 
                  cost=c(0.001, 0.01, 0.1, 1, 5, 10, 100, 1000),
                  gamma=c(0.5, 1, 2, 3, 4), scale=FALSE)
plot(tObj, xlab=expression(gamma), ylab="损失惩罚参数C",
     main="不同参数组合下的预测错误率", 
     nlevels=10, color.palette=terrain.colors)
BestSvm <- tObj$best.model
yPred <- predict(BestSvm, data_test)
(Err <- (sum(ConfM)-sum(diag(ConfM)))/sum(ConfM))   # 0.08
说明：
  • 径向基核(RBF Kernel)可以处理线性不可分问题
  • gamma控制单个样本的影响范围：gamma越大→决策边界越弯曲→过拟合风险越大
  • plot(tObj)生成参数网格的热力图，颜色越深表示误差越低
  • 非线性SVM(径向基核)的预测误差(0.08)通常低于线性SVM

6关联规则分析

来源：Lec10关联规则.pdf · arules包实现Apriori算法挖掘关联规则
Apriori算法核心函数arules
# 读入交易数据
read.transactions(file=文本文件名, format=组织形式, 
                   cols=数值或字符向量, sep=分割符)

# 挖掘频繁项集并生成关联规则
apriori(data=transactions类对象, parameter=NULL, appearance=NULL)

# 查看和筛选规则
size(x=关联规则对象)              # 规则条数
sort(x=关联规则对象, decreasing=TRUE, by=排序依据)  # 排序
subset(x=关联规则对象, subset=条件)                  # 筛选

# 可视化
plot(x=规则对象, method="graph", control=list(main=标题))
说明：
  • apriori(parameter=list(supp=0.001, conf=0.5)) 设定最小支持度和最小置信度
  • sort(by="lift") 按提升度排序找到最有意义的规则
  • subset(rules, lhs %in% "牛奶") 筛选前件包含"牛奶"的规则
  • 序列关联用cspade()：先read_baskets()读入含sequenceID的数据

7时间序列与机器学习

来源：8 第八章 时间序列与机器学习.pdf · 用神经网络/回归树/随机森林预测上证指数收益率和VIX恐慌指数
数据准备：计算对数收益率 + 划分训练集NTS / neuralnet
data <- read.csv("上证指数月度.csv", header=TRUE)
rate <- diff(log(data[,2]))          # 对数差分 = 对数收益率
require(NTS)
set.seed(12345)
m1 <- NNsetting(rate, nfore=30, lags=c(1:3))  # 滞后3阶，预测30期
X <- m1$X; y <- m1$y            # 训练集特征/目标
predX <- m1$predX; predY <- m1$predY  # 测试集特征/目标
df <- data.frame(cbind(y, X))
colnames(df) <- c("rate", "rate1", "rate2", "rate3")
说明：
  • diff(log(x)) 对数差分=对数收益率，常用于金融时间序列的平稳化
  • NNsetting() 将时序数据组织为监督学习格式（滞后特征+预测目标），并划分训练/测试集
  • lags=c(1:3) 用前3期的值预测当期（自回归阶数=3）
神经网络预测时间序列neuralnet / nnet
library(neuralnet)
net <- neuralnet(rate~rate1+rate2+rate3, df, hidden=1,
                  threshold=0.005, learningrate=0.1, 
                  err.fct="sse")
plot(net)
pm3 <- predict(net, newdata=predX)   # 测试集预测
er2 <- pm3 - predY                   # 预测误差
sqrt(mean(er2^2))                      # RMSE: 4.82
mean(abs(er2))                         # MAE:  3.76
说明：
  • 神经网络用于时间序列预测时，输入为滞后值，输出为当期值
  • err.fct="sse" 回归问题使用误差平方和(Sum of Squared Errors)
  • 评估指标：RMSE(均方根误差)=sqrt(mean((pred-actual)²))、MAE(平均绝对误差)
回归树预测时间序列 + 交叉验证剪枝tree
require(tree)
t1 <- tree(rate~., data=data.frame(X))
summary(t1)       # 11个叶节点，残差均方46.44
plot(t1); text(t1, pretty=0)

# 交叉验证选择最佳叶节点数
cv.t1 <- cv.tree(t1)
plot(cv.t1$size, cv.t1$dev, type="b")   # 偏差随叶节点数变化
prune.t1 <- prune.tree(t1, best=5)      # 剪到5个叶节点最优
plot(prune.t1); text(prune.t1, pretty=0)
prun <- predict(prune.t1, newdata=data.frame(predX))
per <- predY - prun
mean(abs(per))                           # 评估绝对误差
说明：
  • tree() 建立回归树（注意：不是rpart！tree包使用不同的算法）
  • cv.tree() 交叉验证确定最优树大小，plot画偏差~节点数
  • prune.tree(best=5) 保留5个叶节点
随机森林预测VIX恐慌指数randomForest / nnet / tseries
mydata <- read.csv("VIX恐慌指数历史数据10年.csv", header=T)
dvix <- mydata[,7]
date <- as.Date(mydata[,1], "%m/%d/%Y")
plot(date, dvix, type="l", ylab="DVIX")

library(tseries)
require(NTS)
m1 <- NNsetting(dvix, nfore=252, lags=c(1:9))   # 滞后9阶

# 线性回归 (baseline)
m2 <- lm(y~-1+X)
yhat <- predX %*% matrix(m2$coefficients, 9, 1)
er1 <- yhat - predY   # RMSE: 3.27

# 神经网络 nnet
require(nnet)
m3 <- nnet(X, y, size=1, skip=T, linout=T, maxit=1000)
pm3 <- predict(m3, newdata=predX)     # RMSE: 3.27

# 随机森林 Bagging (mtry=9→使用全部特征)
rf2 <- randomForest(y~., data=data.frame(X), mtry=9, ntree=1000)
prf2 <- predict(rf2, newdata=data.frame(predX))   # RMSE: 3.25

# 随机森林 (mtry=3→每节点随机选3个特征)
rf3 <- randomForest(y~., data=data.frame(X), mtry=3, ntree=1000)
prf3 <- predict(rf3, newdata=data.frame(predX))   # RMSE: 3.24
说明：
  • 对比四种时间序列预测方法：线性回归(baseline)、nnet、Bagging(mtry=9)、随机森林(mtry=3)
  • nnet中的特殊参数：skip=T添加输入到输出的直接连接(跳跃连接)，linout=T输出层用线性函数(回归)
  • mtry=9→Bagging：每个节点使用全部9个特征=退化为Bagging
  • mtry=3→随机森林：每节点随机选3个≈√9≈3
  • 预测精度的微小差异反映了不同模型的适用性
