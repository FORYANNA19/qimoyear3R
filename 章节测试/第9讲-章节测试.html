一、单项选择题（每题5分，共25分）

1. 感知机模型属于以下哪种类型的分类模型？

答案：B

感知机直接学习决策边界 w·x+b=0，是典型的判别模型。

2. 感知机能够收敛（找到完全正确分类的超平面）的前提条件是什么？

答案：C

感知机收敛的前提是数据线性可分。若线性不可分，算法不会收敛（不断振荡）。

3. 感知机的激活函数通常采用：

答案：C

感知机使用符号函数 sign(w·x+b) 将结果映射为 +1 或 -1。

4. 感知机的损失函数 L(w,b) 本质上是在最小化什么？

答案：B

L(w,b) = -Σ yᵢ(w·xᵢ+b)，衡量的是误分类点到超平面的总距离，不是误分类个数。

5. 感知机与逻辑回归最本质的区别在于：

答案：A

感知机输出的是硬标签 +1/-1，逻辑回归输出的是概率 P(y=1|x)。两者都可以做二分类和多分类（通过 One-vs-Rest 等策略），都使用梯度下降。

二、填空题（每题4分，共8分）

6. 感知机学习算法属于 随机梯度下降法，通过对 误分类点 的梯度下降来极小化损失函数。

7. 万能近似定理指出，仅需 一个（1个） 隐藏层（宽度足够），且激活函数是 非多项式的，就可近似任何连续函数。

三、简答题（每题8分，共8分）

8. 感知机学习算法的权重更新公式为 w ← w + η yᵢ xᵢ。请从几何角度解释该更新的含义，并说明为什么感知机的解不唯一。

参考答案：

几何解释：当点 xᵢ 被误分类时（位于超平面的错误一侧），更新 w ← w + η yᵢ xᵢ 使超平面的法向量向该误分类点的方向（yᵢ=+1 时同向，yᵢ=-1 时反向）调整，从而使分离超平面向误分类点旋转靠近，最终越过该点使其被正确分类。该更新本质上是在减少误分类点到超平面的距离。

解不唯一的原因：

  • 初值选择不同 → 不同最终解
  • 误分类点选取顺序不同 → 不同迭代路径 → 不同分离超平面
  • 感知机只要求将数据「分开」，不要求最大化间隔，所以存在无穷多个合法解（支持向量机 正是通过最大化间隔选择了唯一解）

四、计算题（每题10分，共10分）

9. 已知正实例点 x₁=(3,3)ᵀ, x₂=(4,3)ᵀ，负实例点 x₃=(1,1)ᵀ。采用感知机学习算法，初始 w₀=(0,0)ᵀ, b₀=0, η=1。

(1) 使用 x₁ 作为第一个误分类点，写出第一次更新后的 w₁ 和 b₁。

(2) 写出此时得到的线性模型，并验证 x₃ 是否被正确分类。

参考答案：

(1) 对 x₁=(3,3)ᵀ, y₁=+1：y₁(w₀·x₁ + b₀) = 1×(0+0) = 0 ≤ 0，误分类。

w₁ = w₀ + η·y₁·x₁ = (0,0)ᵀ + 1×1×(3,3)ᵀ = (3,3)ᵀ

b₁ = b₀ + η·y₁ = 0 + 1×1 = 1

(2) 线性模型：w₁·x + b₁ = 3x⁽¹⁾ + 3x⁽²⁾ + 1

验证 x₃=(1,1)ᵀ, y₃=-1：y₃(w₁·x₃ + b₁) = -1×(3×1 + 3×1 + 1) = -1×7 = -7 ≤ 0，仍误分类，需要继续更新。

五、代码分析题（12分）

10. 阅读以下R语言代码，回答问题：

library(nnet)

data(iris)

# 训练单隐藏层神经网络
set.seed(123)
model <- nnet(Species ~ ., data = iris,
              size = 5,
              decay = 0.01,
              maxit = 200,
              linout = FALSE)

print(summary(model))

# 预测并评估
pred <- predict(model, iris, type = "class")
conf <- table(iris$Species, pred)
accuracy <- sum(diag(conf)) / sum(conf)
print(paste("准确率:", round(accuracy * 100, 2), "%"))

(1) size = 5 的含义是什么？如果改为 size = 2 或 size = 20，模型会发生什么变化？（4分）

(2) linout = FALSE 的含义是什么？如果改为 linout = TRUE，网络的性质会发生什么变化？（4分）

(3) decay = 0.01 参数的作用是什么？（4分）

参考答案：

(1) size = 5 表示隐藏层有5个神经元（节点）。若改为 size = 2，隐藏层仅有2个节点，模型表达能力降低，可能导致欠拟合（无法捕捉复杂模式）。若改为 size = 20，隐藏层有20个节点，模型变得非常复杂，参数大量增加，可能过拟合训练数据，泛化能力下降。隐藏层神经元数量是神经网络最重要的超参数之一。

(2) linout = FALSE 表示输出层使用非线性激活（默认是逻辑/Sigmoid函数），适合分类任务。若改为 linout = TRUE，输出层使用线性激活函数（恒等函数），适合回归任务。对于多分类问题（iris有3类），linout=FALSE 时 nnet 使用Softmax输出。

(3) decay = 0.01 是权重衰减（Weight Decay）正则化参数，相当于L2正则化系数λ。它在损失函数中加入惩罚项 λΣw²，防止权重过大，从而缓解过拟合。值越大，权重衰减越强，模型越简单；值越小，模型越自由。它是神经网络中最重要的正则化手段之一。
