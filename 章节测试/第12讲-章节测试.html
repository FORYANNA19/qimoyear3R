一、选择题（每题 5 分，共 30 分）

1支持向量机 的核心目标是寻找一个超平面，该超平面满足什么条件？
答案：C
解析：支持向量机 的基本思想是求解能正确划分训练数据集且使得几何间隔最大的分离超平面。Margin 为 \(2/\|w\|\)，最大化间隔等价于最小化 \(\|w\|^2/2\)。
2在训练 支持向量机 之前，对数据进行特征缩放（如标准化）是否必要？
答案：A
解析：支持向量机 的目标函数涉及距离/内积计算，量纲不一会导致模型偏向于取值范围大的特征，因此标准化非常必要，无论是线性核还是高斯核。
3在线性可分 支持向量机 中，最终决定分类超平面的参数 w 和 b 仅取决于：
答案：C
解析：\(w^* = \sum \alpha_i^* y_i x_i\)，只有 \(\alpha_i > 0\) 的样本（即支持向量）对 w 有贡献。移动或去掉非支持向量不影响最优解。
4在 支持向量机 对偶问题中，如果一个样本点的拉格朗日乘子 \(\alpha_i = 0\)，这意味着：
答案：B
解析：由 KKT条件 互补松弛条件，\(\alpha_i = 0\) 时该点满足 \(y_i(w\cdot x_i+b) \ge 1\)，即被正确分类且不在间隔边界上，对 w 和 b 的求解无贡献。
5当数据不是完全线性可分时，支持向量机 引入"松弛变量"和惩罚参数 C。如果 C 设置得非常大，模型会倾向于：
答案：A
解析：C 越大，对误分类惩罚越重，模型就会竭力减少错误，导致间隔变窄、对训练数据过拟合。C→∞ 退化为硬间隔 支持向量机。
6关于核函数（Kernel Trick），下列说法正确的是：
答案：C
解析：核技巧的核心优势就是在低维空间计算核函数 \(K(x,z)\)，从而隐式地等价于高维特征空间中的内积 \(\phi(x)\cdot\phi(z)\)，避免显式高维映射的计算灾难。

二、填空题（每题 4 分，共 12 分）

7在 支持向量机 中，分类间隔（Margin）的数学表达式为 ______，最大化间隔等价于最小化 ______。
答案：\(\text{margin} = \frac{2}{\|w\|}\)，最小化 \(\frac{1}{2}\|w\|^2\)
8软间隔 支持向量机 的对偶问题中，拉格朗日乘子 \(\alpha_i\) 的取值范围是 ______（硬间隔中为 \(\alpha_i \ge 0\)），其中 C 是 ______ 参数。
答案：\(0 \le \alpha_i \le C\)；惩罚（或调节）参数
9常用的核函数中，______ 核函数可以将原始数据映射到无穷维特征空间。核函数 \(K(x,z)\) 是正定核的充要条件是：其对应的 Gram 矩阵是 ______ 矩阵。
答案：高斯核（径向基函数 (RBF)）；半正定

三、简答题（每题 10 分，共 20 分）

10简述硬间隔 支持向量机 与软间隔 支持向量机 的区别，并说明软间隔中惩罚参数 C 的作用。
参考答案： ① 适用数据不同：硬间隔 支持向量机 要求数据严格线性可分；软间隔 支持向量机 适用于广义线性可分数据（存在异常值/噪声）。 ② 约束条件不同：硬间隔要求 \(y_i(w\cdot x_i+b) \ge 1\)；软间隔放宽为 \(y_i(w\cdot x_i+b) \ge 1-\xi_i\)，引入了松弛变量 \(\xi_i \ge 0\)。 ③ 优化目标不同：硬间隔为 \(\min \frac{1}{2}\|w\|^2\)；软间隔为 \(\min \frac{1}{2}\|w\|^2 + C\sum\xi_i\)，多了对误分类的惩罚项。 ④ C 的作用：C 是调节间隔最大化和误分类惩罚之间权衡的参数。C 越大，对误分类惩罚越重，模型倾向减少错误但间隔变窄（可能过拟合）；C 越小，越容忍错误，间隔更宽（可能欠拟合）。硬间隔是 C→∞ 的特例。 ⑤ 对偶形式：硬间隔中 \(\alpha_i \ge 0\)；软间隔中 \(0 \le \alpha_i \le C\)。
11什么是核技巧（Kernel Trick）？为什么 支持向量机 中使用核技巧能够高效处理非线性分类问题？
参考答案： ① 定义：核技巧是指通过非线性变换 \(\phi(x)\) 将输入空间映射到高维特征空间，在特征空间中学习线性 支持向量机。但由于 支持向量机 对偶问题中只出现样本间的内积 \(x_i\cdot x_j\)，因此只需定义核函数 \(K(x_i, x_j) = \phi(x_i)\cdot\phi(x_j)\) 即可隐式完成映射，无需显式计算 \(\phi(x)\)。 ② 为何高效：核函数在低维空间计算，结果等价于高维空间的内积，避免了"维度灾难"。例如高斯核 \(K(x,z)=\exp(-\|x-z\|^2/2\sigma^2)\) 对应的 \(\phi(x)\) 是无穷维的，但计算仍在原空间完成。 ③ 步骤：将对偶问题中的内积 \(x_i\cdot x_j\) 替换为 \(K(x_i, x_j)\)；决策函数变为 \(f(x) = \text{sign}(\sum \alpha_i^* y_i K(x_i, x) + b^*)\)。 ④ 条件：核函数必须为正定核，即其 Gram 矩阵为半正定矩阵。

四、计算题（共 20 分）

12 设有三个样本点：\(x_1 = (0,0),\ y_1 = +1\)；\(x_2 = (2,2),\ y_2 = +1\)；\(x_3 = (2,0),\ y_3 = -1\)。 它们线性可分。
(1) 写出硬间隔 支持向量机 的原始优化问题。
(2) 求出最优超平面 \(w^*\) 和 \(b^*\)。
(3) 给出分类决策函数和间隔宽度。
参考答案： (1) 原始问题： \(\min_{w,b} \frac{1}{2}(w_1^2 + w_2^2)\) s.t. \(y_i(w_1 x_{i1} + w_2 x_{i2} + b) \ge 1,\ i=1,2,3\) 代入三个样本点： ① \(+1 \cdot (0\cdot w_1 + 0\cdot w_2 + b) \ge 1 \Rightarrow b \ge 1\) ② \(+1 \cdot (2w_1 + 2w_2 + b) \ge 1 \Rightarrow 2w_1 + 2w_2 + b \ge 1\) ③ \(-1 \cdot (2w_1 + 0\cdot w_2 + b) \ge 1 \Rightarrow -2w_1 - b \ge 1 \Rightarrow 2w_1 + b \le -1\) 观察可知：\(x_1\) 和 \(x_3\) 最接近，直觉上支持向量应为 (0,0) 和 (2,0)。 由几何直觉，分离超平面应为 x₁=1 这条垂直线，即 \(w = (1, 0), b = -1\)。 验证：\(w = (1,0), b = -1\) ① \(y_1(w\cdot x_1+b) = 1 \cdot (0+0-1) = -1 \ge 1\)？不成立！ 重新构造：最优超平面垂直于 (0,0) 和 (2,0) 的连线。 重新分析：支持向量应为 (0,0) 和 (2,0)，它们之间的垂直平分线是 x₁=1。 超平面形式：\(x_1 - 1 = 0\)，即 \(w_1 = 1, w_2 = 0, b = -1\)。 代入检查约束： ① \(y_1(w\cdot x_1+b) = +1 \cdot (1\times0 + 0\times0 - 1) = -1\) ❌ 不满足 ≥1 分析：点 (0,0) 为正例，代入 x₁=1 直线得 0-1=-1 < 0，说明其被分到了负类一侧！这不对。 重新观察数据：(0,0)要分到正类，(2,0)分到负类，因此决策边界应在 x₁ ∈ (0,2) 之间。 设超平面为 \(w_1 x_1 + w_2 x_2 + b = 0\)，支持向量为 (0,0) 和 (2,0)。 由 KKT条件 条件：\(\alpha_1(y_1(w\cdot x_1+b)-1) = 0\)，且 \(y_1(w\cdot x_1+b) \ge 1\) 即 \(+1\cdot(w_1\cdot 0 + w_2\cdot 0 + b) \ge 1 \Rightarrow b \ge 1\) \(\alpha_3 > 0\) 且 \(-1\cdot(2w_1 + 0 + b) \ge 1 \Rightarrow -2w_1 - b \ge 1\) 由于是支持向量（等号成立）：\(b = 1\)，\(-2w_1 - b = 1 \Rightarrow -2w_1 - 1 = 1 \Rightarrow w_1 = -1\) 验证：\(w = (-1, 0),\ b = 1\) ① \(+1 \cdot (0+0+1) = 1 \ge 1\) ✓（支持向量，等号成立） ② \(+1 \cdot (-2-2+1) = -3\) ❌ 不满足！ 正例 (2,2) 不满足约束？问题出在这里。让我假设 (0,0) 和 (2,2) 是支持向量： 支持向量处等号成立：\(+1\cdot(b) = 1 \Rightarrow b = 1\)；\(+1\cdot(2w_1 + 2w_2 + 1) = 1 \Rightarrow 2w_1 + 2w_2 = 0 \Rightarrow w_1 = -w_2\) 另一侧负例支持向量是 (2,0)：\(-1\cdot(2w_1 + 0 + 1) = 1 \Rightarrow -2w_1 - 1 = 1 \Rightarrow w_1 = -1\) 所以 \(w_1 = -1, w_2 = 1, b = 1\)。 验证：\(w = (-1, 1),\ b = 1\) ① \(+1\cdot(0+0+1) = 1 \ge 1\) ✓ ② \(+1\cdot(-2+2+1) = 1 \ge 1\) ✓ ③ \(-1\cdot(-2+0+1) = -1\cdot(-1) = 1 \ge 1\) ✓ 所有三个点都恰好满足等式约束，全部是支持向量！ (2) 最优解：\(w^* = (-1, 1),\ b^* = 1\)（或等价地，\(\tilde{w}^* = (1, -1),\ \tilde{b}^* = -1\) 因为乘以 -1 等价） 规范形式一般取法向量：\(\tilde{w} = (1, -1),\ \tilde{b} = 1\)，超平面为 \(x_1 - x_2 + 1 = 0\)。 (3) 分类决策函数：\(f(x) = \text{sign}(-x_1 + x_2 + 1)\) 或 \(f(x) = \text{sign}(x_1 - x_2 - 1)\) 间隔宽度：\(\text{margin} = \frac{2}{\|w\|} = \frac{2}{\sqrt{(-1)^2 + 1^2}} = \frac{2}{\sqrt{2}} = \sqrt{2}\)

五、代码分析题（共 18 分）

13 阅读以下使用 e1071 包训练 支持向量机 的 R 代码，回答后面问题： 
library(e1071)
data(iris)
model <- svm(Species ~ ., data = iris,
             kernel = "radial", cost = 1, 
             gamma = 0.5, scale = TRUE)
cat("支持向量数量:", nrow(model$SV), "\n")
pred <- predict(model, iris)
conf_matrix <- table(iris$Species, pred)
accuracy <- sum(diag(conf_matrix)) / sum(conf_matrix)
cat(sprintf("准确率: %.2f%%\n", accuracy * 100))

问题：

  1. kernel = "radial" 使用的是什么核函数？在期末考试中，哪几种核函数可以用于非线性建模？（4分）
  2. scale = TRUE 的作用是什么？为什么在训练支持向量机前要做这一步？（4分）
  3. cost = 1 参数控制什么？如果 cost 设置得过大或过小，分别会导致什么问题？（5分）
  4. nrow(model$SV) 返回的支持向量数量通常应该 多于 还是 少于 训练样本数？为什么？（5分）

参考答案：

(1) kernel="radial" 使用的是径向基函数核。可以用于非线性建模的核函数包括：径向基函数核、多项式核、Sigmoid核。线性核只能用于线性可分问题。期末考试中会考"以下哪个核函数不适合用于支持向量机非线性建模"——答案选择要对比各核的性质。

(2) scale=TRUE 对数据进行标准化（均值为0，标准差为1）。支持向量机依赖距离度量，如果不做标准化，取值范围大的特征（如收入0~100000）会主导距离计算，使得模型偏向大方差特征。期末考试中这个知识点常客："在训练支持向量机之前标准化是否必要——非常必要"。

(3) cost 是软间隔惩罚参数 C，控制对误分类的容忍度。cost 过大 → 过拟合（强制完美分类，间隔变小，泛化差）；cost 过小 → 欠拟合（容忍大量误分，间隔大但分类不准）。

(4) 少于训练样本数。支持向量机 的解具有稀疏性——只有落在间隔边界上或间隔内的样本（支持向量）才影响决策边界，其余样本不出现在解的表达式中。支持向量通常只占训练样本的一小部分，这是一个考点。
