第2讲 章节测试：模型评估与选择

共 16 道题 · 单选题7 + 填空题3 + 简答题3 + 计算题2 + 编程题1

一、选择题（7题）

1. 在机器学习三要素中，使用"梯度下降法"求解逻辑回归的参数属于哪一个要素？（课堂原题）
A. 模型
B. 算法
C. 特征
D. 策略
答案：B
三要素中：模型指定假设空间（如逻辑回归函数形式），策略定义损失函数（如对数似然损失），算法指定如何求解最优参数（如梯度下降法）。
2. 在模型选择中，如果模型在训练集上表现极好，但在测试集上表现很差，这种现象通常称为什么？（课堂原题）
A. 欠拟合
B. 过拟合
答案：B. 过拟合
过拟合：训练误差低但泛化能力差（测试误差高），模型过度学习了训练数据中的噪声。
3. 在二分类问题中，如果一个模型将所有样本都预测为正例，那么该模型的哪个指标一定很高？（课堂原题）
A. 查全率 (Recall)
B. 查准率 (Precision)
C. F1
答案：A. 查全率
全部预测为正例时，FN=0，所以 Recall = TP/(TP+FN) = 1。但 FP 很大，Precision = TP/(TP+FP) 很低。
4. 以下关于查准率 (Precision) 和查全率 (Recall) 的说法，错误的是：
A. 查准率和查全率是一对矛盾的度量
B. F1值是查准率和查全率的调和平均
C. Fβ 中 β > 1 表示查准率有更大影响
D. P-R曲线完全被包住时，被包住的学习器性能更差
答案：C
Fβ 中 β > 1 表示查全率 (Recall) 有更大影响（如逃犯检索）；β < 1 才是查准率有更大影响（如商品推荐）。
5. 自助法 (Bootstrap) 中，训练集和原始数据集D规模相同，但约有___的样本不会出现在训练集中：
A. 50%
B. 25%
C. 36.8%
D. 10%
答案：C. 36.8%
推导：每次不被抽到的概率为 (1-1/m)，m次后 $(1-1/m)^m \to 1/e \approx 0.368$。
6. 下列哪个是留一法 (Leave-One-Out) 的正确描述？
A. 将数据集分为两部分，一部分训练，一部分测试
B. k折交叉验证中 k=N 时的特殊情况
C. 有放回地抽取样本构成训练集
D. 训练集和测试集按 4:1 划分
答案：B
留一法是k折交叉验证的特例（k=N），每次留一个样本作测试，其余用于训练，评估准确但计算复杂度高。
7. 关于AUC，以下说法正确的是：
A. AUC = 0.8 说明模型优于随机猜测，有预测价值
B. AUC = 0.4 比 AUC = 0.5 的模型更差，且无法改善
C. AUC 值越大一定说明分类效果越好
D. AUC = 0.5 表示模型分类完全正确
答案：A
A：正确，0.5 < AUC < 1 优于随机；B：错误，AUC < 0.5 反着预测即可优于随机；C：AUC在不同问题间不可直接比较绝对大小；D：错误，AUC = 0.5 等同于随机猜测。

二、填空题（3题）

8. 泛化误差可分解为三项之和：$\text{泛化误差} = \text{Bias}^2 + \text{Var} + \boxed{\qquad}$
答案：$\sigma^2$（不可约误差/噪声方差）
$\mathbb{E}[(Y-\hat{f})^2] = \text{Bias}^2 + \text{Var} + \sigma^2$。前两项可通过改进模型控制，$\sigma^2$ 是数据固有的随机误差，任何模型都无法消除。
9. 结构风险由经验风险和正则化项构成：$R_{\text{struct}}(f) = R_{\text{emp}}(f) + \boxed{\qquad}$，其中调节参数用 $\boxed{\qquad}$ 表示。
答案：$\lambda \cdot J(f)$；$\lambda$
$J(f)$ 衡量模型复杂度，$\lambda$ 是调节参数 (tuning parameter)，用于权衡经验风险和模型复杂度。
10. 混淆矩阵中，TP表示真实为    且预测为    的样本数；FP表示真实为    且预测为    的样本数。
答案：正例/正例(Positive/Positive)；负例/正例(Negative/Positive)
P=正例（好西瓜），N=负例（坏西瓜）。T=预测正确，F=预测错误。TP：正确标记为正；FP：错误标记为正。

三、简答题（3题）

11. 简述留出法、k折交叉验证法和自助法三者的区别与适用场景。
答案要点：
  1. 留出法：直接将数据划分为训练/测试集，最简单。适用于样本量大的情形。缺点是评估结果受划分方式影响大。
  2. k折交叉验证：将数据分k份，轮换k次训练/测试取均值。评估更稳定可靠，但计算量比留出法大k倍。适用于中等规模数据。
  3. 自助法：有放回抽样，约36.8%样本作测试集。适用于数据集较小的情况。但改变了数据分布，训练样本充足时不推荐。
12. 从偏差-方差分解的角度解释：为什么增加训练样本数量可以有效缓解过拟合，但对欠拟合帮助不大？
答案要点：

过拟合 (High Variance)：模型方差大，即不同训练集导致模型预测波动剧烈。增加样本量可以稳定模型估计，降低方差 → 有效。

欠拟合 (High Bias)：模型偏差大，即模型本身的拟合能力不足以捕捉数据规律（如用线性模型拟合非线性数据）。偏差来源于模型结构限制，不随样本量增加而减小 → 帮助不大。需要增加模型复杂度或引入更多特征。

直观理解：给一个近视的人（低复杂度模型）看再多数据，他也看不清远处的细节（高偏差）。

13. 岭回归的偏差-方差分解中，$\lambda$ 对偏差和方差分别有何影响？为什么岭回归可能获得比最小二乘更低的预测误差？
答案要点：

$\lambda$ 增大 → 系数向零压缩 → 偏差增大，方差降低。

$\lambda$ 减小 → 趋近最小二乘 → 偏差减小（趋于0），方差增大。

岭回归可能获得更低预测误差的原因：虽然引入了偏差，但如果方差降低的幅度大于偏差平方增加的幅度，总预测误差 $\text{Bias}^2 + \text{Var}$ 会低于最小二乘（无偏但方差大）。这就是"以偏差换方差"的策略。

四、计算题（2题）

14. 某二分类模型的测试结果混淆矩阵如下。请计算：准确率 (Accuracy)、查准率 (Precision)、查全率 (Recall)、F1值。
预测为正预测为负
真实为正TP = 80FN = 20
真实为负FP = 30TN = 70
答案：

总样本 = 80 + 20 + 30 + 70 = 200

准确率 Accuracy = (80+70)/200 = 75%

查准率 Precision = 80/(80+30) = 80/110 ≈ 72.7%

查全率 Recall = 80/(80+20) = 80/100 = 80%

F1 = 2 × 0.727 × 0.8 / (0.727 + 0.8) = 1.1632 / 1.527 ≈ 0.762

15. 某在线问卷调查场景：向1000人邮寄问卷，baseline回收率为20%（即200人回复）。利用模型筛选出100人，其中60人回复。请计算Lift提升度。
答案：Lift = 3

正例基线比例 (baseline) = 200/1000 = 20%

模型预测正例的准确率 = 60/100 = 60%

Lift = 60% / 20% = 3

含义：运用模型后效果提升了3倍。Lift > 1 说明模型有效。

五、代码分析题（1题，12分）

16. 阅读以下R语言代码，回答问题：
参考答案： 
library(caret)
library(e1071)

# 设置10折交叉验证
ctrl <- trainControl(method = "cv", number = 10)

# 训练逻辑回归模型
model <- train(Species ~ ., data = iris,
               method = "glm",
               trControl = ctrl)

print(model)
print(model$results)

(1) method = "cv", number = 10 的含义是什么？这样设置后模型训练过程是如何进行的？（4分）

method = "cv" 表示采用交叉验证（Cross-Validation），number = 10 表示进行10折交叉验证。训练过程：将训练数据随机等分为10份，每次用9份训练、1份验证，轮换10次，最终取10次验证结果的平均值作为模型性能估计。

(2) Species ~ . 公式的含义是什么？. 代表什么？（4分）

Species ~ . 表示以Species为响应变量（目标变量），以数据框中所有其他列为预测变量。. 是R公式中的简写符号，代表"除响应变量外的全部变量"。等价于 Species ~ Sepal.Length + Sepal.Width + Petal.Length + Petal.Width。

(3) model$results 会输出什么内容？其中哪个参数反映了交叉验证的效果？（4分）

model$results 输出模型的交叉验证详细结果，包括每个超参数组合下的Accuracy、Kappa等指标。对于 glm 方法，由于无超参数需要调优，它会显示单行结果。Accuracy 列反映了交叉验证对模型预测准确率的估计，是评估模型泛化能力的关键指标。
