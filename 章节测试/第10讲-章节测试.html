一、单项选择题（每题5分，共30分）

1. 感知机模型 f(x) = sign(w·x + b) 中，sign 函数的输出范围是：

答案：C

sign 函数输出为 +1 或 -1，对应正类和负类。{0, 1} 是阶跃函数，(0, 1) 是 Sigmoid激活函数。

2. 感知机能够收敛的前提条件是什么？

答案：C

感知机收敛的充要条件是数据集线性可分（Linearly Separable）。

3. 感知机定义的损失函数 L(w, b) 本质上是在最小化什么？

答案：B

L(w,b) = −Σ yᵢ(w·xᵢ + b), xᵢ∈M，本质是误分类点到超平面的总距离。错误个数不可导，无法直接优化。

4. 在感知机算法中，学习率 η（0 < η ≤ 1）对最终是否收敛有何影响？

答案：C

η 仅影响收敛速度/迭代次数。只要数据线性可分，感知机总会在有限步内收敛。

5. 关于激活函数 ReLU激活函数 (f(x)=max(0,x)) 相比 Sigmoid激活函数，以下哪项是其主要优点？

答案：C

ReLU激活函数 在 x>0 时导数为 1，梯度不会衰减。Sigmoid激活函数 导数最大仅 0.25，深层连乘后梯度接近 0。ReLU激活函数 在 x=0 处不可导（实际用次梯度）。

6. 万能近似定理告诉我们，一个至少有一个隐藏层、使用非多项式激活函数的 多层感知机 (MLP) 可以：

答案：B

万能近似定理：仅需 1 个隐藏层 + 足够多神经元 + 非多项式激活函数，即可以任意精度逼近任何连续函数。

二、填空题（每题5分，共15分）

1. 感知机模型 f(x) = sign(w·x + b) 中，w 称为 权值（或权值向量），b 称为 偏置，w·x + b = 0 在特征空间 ℝⁿ 中表示一个 超平面。

答：权值；偏置；超平面。w 是超平面的法向量，b 是截距。

2. 反向传播 (BP) 算法分为两个阶段：前向传播 计算损失和 反向传播 计算梯度。更新权重的核心数学工具是微积分中的 链式法则。

答：前向传播；反向传播；链式法则 (Chain Rule)。

3. 传统的 Sigmoid 函数在深层网络中会引起 梯度消失 问题，原因是其导数最大值为 0.25，在深层连乘中指数衰减至接近 0。

答：梯度消失 (Vanishing Gradient)；0.25。Sigmoid激活函数 导数 max = f(0)(1-f(0)) = 0.5×(1-0.5) = 0.25。

三、简答题（每题10分，共20分）

1. 简述 M-P 神经元模型的基本工作原理，并说明感知机如何基于此模型构建二分类器。

参考答案：

  • M-P 神经元模型：接收来自 n 个其他神经元的输入信号，通过带权重的连接传递。总输入值 Σ wᵢxᵢ 与阈值 θ 比较，经激活函数 f 处理后产生输出 y = f(Σ wᵢxᵢ - θ)。
  • 感知机构建二分类器：感知机由两层神经元组成——输入层接收特征，输出层是 M-P 神经元。激活函数采用符号函数 sign，输出 ±1 对应正/负二类。感知机在特征空间中寻找分离超平面 w·x + b = 0 进行二分类，是一种线性判别模型。
  • 学习过程：通过随机梯度下降法 (随机梯度下降 (SGD)) 最小化误分类点到超平面的总距离，不断更新 w 和 b，直至所有样本正确分类。

2. 简述 Standard 反向传播 (随机梯度下降 (SGD))、Accumulated 反向传播 (Batch) 和 Mini-batch 反向传播 三种梯度下降方式在更新频率、收敛路径和适用场景方面的主要区别。

参考答案：

  • Standard 反向传播 (随机梯度下降 (SGD))：每个样本更新一次 → 更新频率极高，收敛路径剧烈震荡。优点是跳出局部最优能力强；缺点是无法利用 GPU 并行性，计算开销大，收敛精度差。
  • Accumulated 反向传播 (Batch)：每个 Epoch 更新一次（需扫描完所有样本）→ 更新频率极低。梯度估计准确，收敛路径平滑；但内存需求极高（需载入全量数据），效率极低，易陷入局部最优。
  • Mini-batch BP：折中方案，每处理一个小批量（如 32、64、128 样本）更新一次。利用 GPU 矢量化计算，平衡了内存、速度和泛化性能，是现代深度学习工业标准。

四、计算题（15分）

题目：给定二分类训练数据：正类样本 x₁ = (3, 3)ᵀ, y₁ = +1；负类样本 x₂ = (1, 1)ᵀ, y₂ = -1；负类样本 x₃ = (0, 2)ᵀ, y₃ = -1。

设初始 w = (0, 0)ᵀ, b = 0，学习率 η = 1。采用感知机 随机梯度下降 (SGD) 算法训练，数据按 x₁ → x₂ → x₃ 顺序检查。

请计算前 3 轮迭代（每轮遍历三个样本一次）中每次权重更新的结果。

解答过程：

初始：w = (0, 0), b = 0

第1轮：

  • x₁=(3,3), y₁=+1: w·x₁ + b = 0, y₁×0 = 0 ≤ 0 ✘ 误分类！
    w ← (0,0) + 1×(3,3) = (3, 3)，b ← 0+1 = 1
  • x₂=(1,1), y₂=-1: w·x₂ + b = 3+3+1 = 7, y₂×7 = -7 ≤ 0 ✘ 误分类！
    w ← (3,3) + (-1)×(1,1) = (2, 2)，b ← 1 + (-1) = 0
  • x₃=(0,2), y₃=-1: w·x₃ + b = 0+4+0 = 4, y₃×4 = -4 ≤ 0 ✘ 误分类！
    w ← (2,2) + (-1)×(0,2) = (2, 0)，b ← 0 + (-1) = -1

第2轮：w = (2, 0), b = -1

  • x₁=(3,3): w·x₁ + b = 6+0-1 = 5, y₁×5 = 5 > 0 ✓ 正确，不更新
  • x₂=(1,1): w·x₂ + b = 2+0-1 = 1, y₂×1 = -1 ≤ 0 ✘ 误分类！
    w ← (2,0) + (-1)×(1,1) = (1, -1)，b ← -1 + (-1) = -2
  • x₃=(0,2): w·x₃ + b = 0-2-2 = -4, y₃×(-4) = 4 > 0 ✓ 正确，不更新

第3轮：w = (1, -1), b = -2

  • x₁=(3,3): w·x₁ + b = 3-3-2 = -2, y₁×(-2) = -2 ≤ 0 ✘ 误分类！
    w ← (1,-1) + 1×(3,3) = (4, 2)，b ← -2 + 1 = -1
  • x₂=(1,1): w·x₂ + b = 4+2-1 = 5, y₂×5 = -5 ≤ 0 ✘ 误分类！
    w ← (4,2) + (-1)×(1,1) = (3, 1)，b ← -1 + (-1) = -2
  • x₃=(0,2): w·x₃ + b = 0+2-2 = 0, y₃×0 = 0 ≤ 0 ✘ 误分类！
    w ← (3,1) + (-1)×(0,2) = (3, -1)，b ← -2 + (-1) = -3

3轮结束：w = (3, -1), b = -3。超平面：3x₁ - x₂ - 3 = 0。

验证第4轮：x₁(3,3): 9-3-3=3>0 ✓; x₂(1,1): 3-1-3=-1<0 → y₂×(-1)=1>0 ✓; x₃(0,2): 0-2-3=-5<0 → y₃×(-5)=5>0 ✓ — 全部正确！收敛。

五、代码分析题（每题10分，共20分）

题目1：阅读以下 R 语言感知机实现代码，回答后面的问题。

set.seed(123)
X <- matrix(rnorm(200), ncol=2)
y <- ifelse(X[,1] + X[,2] > 0, 1, -1)

w <- c(0, 0)
b <- 0
eta <- 0.1

for (epoch in 1:10) {
  for (i in 1:nrow(X)) {
    if (y[i] * (sum(w * X[i,]) + b) <= 0) {
      w <- w + eta * y[i] * X[i,]
      b <- b + eta * y[i]
    }
  }
}
cat("权重 w:", w, "\n")
cat("偏置 b:", b, "\n")

问题：

  1. 代码中 sum(w * X[i,]) + b 计算的是什么？为什么要检查 y[i] * (该值) <= 0？（5分）
  2. 如果我们将数据集改为线性不可分的异或（XOR）问题，这段代码会发生什么？学习率 eta 是否影响最终能否收敛？（5分）

参考答案：

(1) sum(w * X[i,]) + b 计算的是线性判别函数值 w·x+b。条件 y[i] * (该值) <= 0 判断样本是否被误分类：当预测值（w·x+b 的符号）与真实标签 y 不一致时，二者乘积为负。这是感知机算法的核心——只在误分类时更新权重。

(2) 对于 XOR 等线性不可分问题，感知机永远不会收敛。error_count 始终 > 0，循环会进行完所有 epoch 后停止。学习率 η 不影响收敛性——对线性可分问题，无论 η 多小，最终都会收敛（Novikoff 定理）；对线性不可分问题，无论 η 多大，都不会收敛。

题目2：阅读以下基于交叉熵损失的 Sigmoid激活函数 感知机 R 语言实现。回答后面的问题。

sigmoid <- function(z) { 1.0 / (1.0 + exp(-z)) }

forward <- function(X, w, b) {
  z <- X %*% w + b
  y_hat <- sigmoid(z)
  return(y_hat)
}

backward <- function(X, y, y_hat, w, b, eta) {
  n <- nrow(X)
  dz <- y_hat - y
  dw <- (1/n) * t(X) %*% dz
  db <- (1/n) * sum(dz)
  w <- w - eta * dw
  b <- b - eta * db
  return(list(w=w, b=b))
}

问题：

  1. 代码中 dz <- y_hat - y 的简洁形式是 Sigmoid激活函数 + 交叉熵损失组合下的特例。如果改用均方误差损失，这个简洁形式是否仍然成立？为什么？（5分）
  2. dw <- (1/n) * t(X) %*% dz 中 t(X) 的作用是什么？为什么需要转置？（5分）

参考答案：

(1) 不成立。dz = y_hat - y 的简洁形式是交叉熵损失 + Sigmoid激活函数 的特例——两者的导数恰好抵消了 Sigmoid激活函数 导数的 f'(z)=f(z)(1-f(z)) 项。如果改用均方误差损失，梯度会多出一个 f'(z) 因子，不仅形式更复杂，还可能导致梯度消失问题。

(2) 梯度 dw 的维度应与 w 一致。X 是 n×d 矩阵，dz 是 n×1 向量。X^T · dz = (d×n)(n×1) = d×1，与 w 维度相同。转置 X 是为了正确完成矩阵乘法，使每个特征的梯度是对应所有样本的加权求和。
