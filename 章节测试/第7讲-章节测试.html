一、单项选择题（每题5分，共20分）

1. AdaBoost 中，弱分类器权重系数 αm 的计算公式为：

答案：C

αm = ½ ln((1-εm)/εm)，其中 εm 为加权分类误差率。

2. 下列关于 AdaBoost 的说法，错误的是：

答案：C

AdaBoost 不采用 Bootstrap 重采样，而是直接调整样本权重。Bootstrap 是 Bagging/Random Forest 的做法。

3. 若某个弱分类器在训练集上的加权错误率 ε = 0.5（二分类问题），则：

答案：B

α = ½ ln(0.5/0.5) = 0。ε=0.5 意味着分类器如同随机猜测，没有学习到任何信息。

4. AdaBoost 与随机森林最大的区别在于：

答案：B

随机森林各树可以并行训练（无依赖），AdaBoost 后一轮依赖前一轮结果（串行）。

二、填空题（每题5分，共10分）

1. AdaBoost 算法中，第 m 轮的分类器权重 αm 与误差率 εm 呈 负相关 关系，εm 越小，αm 越大。

答：负相关；小；大。α = ½ ln((1-ε)/ε) 是 ε 的单调递减函数。

2. AdaBoost 优化的损失函数是 指数损失，其数学形式为 L(y, f(x)) = exp(-y·f(x))。

答：指数损失 (Exponential Loss)；exp(-y·f(x))。

三、简答题（每题10分，共20分）

1. 简述 Bagging（如随机森林）与 Boosting（如 AdaBoost）在训练方式、样本处理、组合方式和优化目标四个方面的主要区别。

参考答案：

  • 训练方式：Bagging 是并行训练（各分类器独立无依赖）；Boosting 是串行训练（后一个依赖前一个）。
  • 样本处理：Bagging 使用 Bootstrap 有放回抽样生成不同训练子集；AdaBoost 不重抽样，而是调整样本权重。梯度提升树 使用全部样本但拟合残差/负梯度。
  • 组合方式：Bagging 采用简单平均（回归）或多数投票（分类）；Boosting 采用加权投票（AdaBoost）或累加（梯度提升树）。
  • 优化目标：Bagging 主要降低方差（Variance）；Boosting 主要降低偏差（Bias）。

2. 简述 AdaBoost 与前向分步加法模型（Forward Stagewise Additive Modeling）之间的关系。

参考答案：

AdaBoost 可以从前向分步加法模型的角度重新解释：

  • 加法模型：f(x) = Σm αm · Gm(x)，即多个基函数的线性组合。
  • 损失函数：采用指数损失 L(y, f(x)) = exp(-y · f(x))。
  • 前向分步：每次只优化当前步的 (αm, Gm)，不回溯调整之前已确定的参数。
  • 推导等价：在指数损失下，前向分步算法的每一步更新恰好等价于 AdaBoost 的权重更新规则，证明了 AdaBoost 本质上是加法模型配合指数损失的前向分步优化。

四、计算题（15分）

题目：某二分类问题有 5 个训练样本，初始权重均为 0.2。第 1 轮迭代后，弱分类器 G1(x) 正确分类了前 4 个样本，误分类了第 5 个样本。请计算：

  1. 第 1 轮的加权分类误差率 ε1
  2. 该分类器的权重系数 α1
  3. 第 5 个样本在第 2 轮的权重（未经归一化前）

解答过程：

(1) ε1 = 0.2（只有第5个样本被误分类，其权重为0.2）

(2) α1 = ½ ln((1-0.2)/0.2) = ½ ln(4) = ½ × 1.3863 = 0.6931

(3) 第5个样本被误分类，设 y5 = +1, G1(x5) = -1（或反过来），则 y5 · G1(x5) = -1。

更新前权重 w1,5 = 0.2，更新后（未归一化）：

w2,5 = 0.2 × exp(-0.6931 × (-1)) = 0.2 × exp(0.6931) = 0.2 × 2 = 0.4

正确分类的前4个样本（未归一化）：0.2 × exp(-0.6931) = 0.2 × 0.5 = 0.1

归一化因子 Z1 = 4×0.1 + 0.4 = 0.8，归一化后第5个样本权重 = 0.4/0.8 = 0.5

五、代码分析题（12分）

题目：阅读以下R语言代码，回答问题。

library(adabag)
library(rpart)

# 加载数据
data(iris)

# 二分类设定：仅取两类
iris2 <- iris[iris$Species != "virginica", ]
iris2$Species <- factor(iris2$Species)

# AdaBoost训练
set.seed(123)
model <- boosting(Species ~ ., data = iris2,
                  mfinal = 50,
                  coeflearn = "Breiman",
                  control = rpart.control(maxdepth = 1))

print(names(model))
print(model$importance)

# 预测
pred <- predict.boosting(model, newdata = iris2)
print(table(pred$class, iris2$Species))

# 查看第10轮的分类器权重
print(paste("第10个弱分类器权重:", model$weights[10]))

问题：

  1. mfinal = 50 的含义是什么？增大 mfinal 对模型有什么影响？（4分）
  2. rpart.control(maxdepth = 1) 为什么将深度设为1？这对应AdaBoost中的什么概念？（4分）
  3. model$weights 向量存储了什么？它的数值范围有什么特点？（4分）

参考答案：

(1) mfinal = 50 表示进行50轮迭代，训练50个弱分类器。增大 mfinal 意味着更多轮Boosting，通常可以进一步降低偏差（纠错机会增多），但计算时间增加且可能过拟合（当弱分类器开始拟合噪声时）。AdaBoost的轮数相当于集成中的弱学习器个数。

(2) maxdepth = 1 将每棵决策树限制为决策树桩（Decision Stump）——只分裂一次即停止。这对应AdaBoost中弱分类器的概念：每个基分类器只需比随机猜测略好即可（弱学习器）。使用深度为1的树是最常见的弱分类器选择，因为它结构简单、方差小。

(3) model$weights 存储每个弱分类器在最终集成中的投票权重 αm，共50个值。权重 α = ½ ln((1-ε)/ε)，其中 ε 是该轮加权误分类率。特点：误差率越小的分类器权重越大（贡献越大）；若 ε = 0.5（随机猜测），α = 0；若 ε → 0，α → +∞。
