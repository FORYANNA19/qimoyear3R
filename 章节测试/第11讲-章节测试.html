第11讲 章节测试：BP反向传播算法

上海对外经贸大学 · 数据挖掘与机器学习  |  满分：100分  |  建议用时：45分钟

一、单项选择题 （每题4分，共20分）

1多层感知机（多层感知机 (MLP)）与单层感知机最本质的区别是什么？
  • A. 多层感知机 (MLP) 拥有更多的输入特征
  • B. 多层感知机 (MLP) 引入了隐藏层（Hidden Layer），从而能够处理线性不可分问题
  • C. 多层感知机 (MLP) 使用了更大的学习率
  • D. 多层感知机 (MLP) 只能处理回归任务，不能处理分类任务
答案：B

多层感知机 (MLP) 引入隐藏层及非线性激活函数，使其能学习非线性决策边界，处理线性不可分问题。

2如果在 多层感知机 (MLP) 中使用线性激活函数（如 \(f(x)=x\)），这个网络等价于什么？
  • A. 深度卷积神经网络
  • B. 支持向量机
  • C. 简单的线性模型
  • D. 逻辑回归模型
答案：C

多层线性变换的复合仍然是线性变换 \(\mathbf{W}_2(\mathbf{W}_1\mathbf{x}) = (\mathbf{W}_2\mathbf{W}_1)\mathbf{x}\)，等价于单层线性模型。

3在 反向传播 算法中，更新权重的核心数学工具是什么？
  • A. 泰勒级数展开
  • B. 蒙特卡洛模拟
  • C. 微积分中的链式法则
  • D. 矩阵的特征值分解
答案：C

反向传播 算法通过链式法则将损失对权重的梯度分解为多个可计算的局部导数的乘积。

4在 反向传播 算法的"反向传播"阶段，主要计算的是什么？
  • A. 每一层神经元的激活值
  • B. 损失函数对网络中各个权值和偏置的梯度
  • C. 输入数据的特征分布
  • D. 模型的预测准确率
答案：B

反向传播阶段从输出层到输入层逐层计算损失对各参数的梯度，前向传播才计算激活值和预测值。

5假设一个 多层感知机 (MLP) 输入层 10 节点，隐藏层 20 节点，输出层 5 节点（不计偏置），共有多少个可训练权重？
  • A. 35
  • B. 150
  • C. 300
  • D. 1000
答案：C

输入→隐层：\(10 \times 20 = 200\)；隐层→输出：\(20 \times 5 = 100\)；合计 300 个权重参数。

6关于 Mini-Batch 梯度下降，以下哪项表述是错误的？
  • A. Mini-Batch 是总体梯度的无偏估计
  • B. Batch Size 增大时梯度估计的方差会降低
  • C. Mini-Batch 的噪声可能导致模型陷入更差的局部极小值
  • D. 现代 GPU 并行计算使 Mini-Batch 获得高吞吐量
答案：C

Mini-Batch 的适度噪声反而帮助模型跳出局部极小值的"浅坑"，找到泛化能力更好的平坦区域，属于噪声正则化效应。

二、填空题 （每题5分，共15分）

7反向传播 算法中的"批量 (Batch)"概念是指：在更新一次参数之前，计算________的________梯度。
答案：多个样本；平均

Mini-Batch 每次从训练集中抽取一小批样本，计算它们的平均梯度来近似总体梯度。

8学习率 \(\eta\) 太大会导致________，\(\eta\) 太小会导致________。
答案：振荡/不收敛；收敛速度过慢

学习率控制参数更新的步长，需要在收敛速度和稳定性之间取得平衡。

9Sigmoid 函数的导数为 \(f'(x) = \)________。该性质在 反向传播 推导中用于简化 \(g_j\) 和 \(e_h\) 的计算。
答案：\(f(x)(1-f(x))\)

利用此性质：\(g_j = f(\beta_j-\theta_j)(1-f(\beta_j-\theta_j))(y_j-\hat{y}_j) = \hat{y}_j(1-\hat{y}_j)(y_j-\hat{y}_j)\)。

三、简答题 （每题10分，共20分）

10简述 反向传播 反向传播算法的两个阶段（前向传播和反向传播）各自完成什么任务。
参考答案：

前向传播 (Forward)：输入信号依次通过各层神经元处理，逐层计算激活值 \(\alpha_h, b_h, \beta_j, \hat{y}_j\)，最终在输出层产生预测值并计算与真实值的损失 \(E_k\)。

反向传播 (Backward)：从输出层开始，将损失沿网络结构反向传播，利用链式法则逐层计算每一层权重和偏置对损失的贡献（即梯度 \(\partial E_k/\partial w, \partial E_k/\partial v, \partial E_k/\partial \theta, \partial E_k/\partial \gamma\)），并据此更新参数。

11解释为什么在 多层感知机 (MLP) 中必须使用非线性激活函数？若全部使用线性激活函数会导致什么问题？
参考答案：

非线性激活函数赋予神经网络学习复杂非线性函数的能力，是处理线性不可分问题的关键。

若全部使用线性激活函数，多层线性变换的复合 \(\mathbf{y} = \mathbf{W}_L(\dots(\mathbf{W}_1\mathbf{x})) = \mathbf{W}_{\text{eff}}\,\mathbf{x}\) 仍为线性变换，等价于一个单层线性模型，丧失了深度网络的表达能力。

四、计算题 （25分）

12给定一个简单的两层网络：输入层 2 个神经元 \((x_1, x_2)\)，隐藏层 2 个神经元（Sigmoid 激活），输出层 1 个神经元（Sigmoid 激活）。已知：

\(\mathbf{x} = [0.5, 0.8]^T\)，真实标签 \(y = 0.3\)，学习率 \(\eta = 0.5\)
输入→隐层权重：\(v_{11}=0.2, v_{12}=0.4, v_{21}=-0.3, v_{22}=0.1\)
隐层阈值：\(\gamma_1=0.1, \gamma_2=-0.2\)
隐→输出权重：\(w_{11}=0.3, w_{21}=-0.5\)
输出层阈值：\(\theta_1=0.2\)
损失函数：\(E = \frac{1}{2}(\hat{y} - y)^2\)

(1) 完成一次前向传播，计算预测值 \(\hat{y}\) 和损失 \(E\)。（10分）
(2) 计算输出层误差 \(g_1\)，并更新 \(w_{11}, w_{21}, \theta_1\)。（15分）

解：

(1) 前向传播

\(\alpha_1 = v_{11}x_1 + v_{21}x_2 = 0.2\times 0.5 + (-0.3)\times 0.8 = 0.1 - 0.24 = -0.14\)
\(\alpha_2 = v_{12}x_1 + v_{22}x_2 = 0.4\times 0.5 + 0.1\times 0.8 = 0.2 + 0.08 = 0.28\)
\(b_1 = \sigma(\alpha_1 - \gamma_1) = \sigma(-0.14 - 0.1) = \sigma(-0.24) \approx 0.4403\)
\(b_2 = \sigma(\alpha_2 - \gamma_2) = \sigma(0.28 - (-0.2)) = \sigma(0.48) \approx 0.6177\)
\(\beta_1 = w_{11}b_1 + w_{21}b_2 = 0.3 \times 0.4403 + (-0.5) \times 0.6177 = 0.1321 - 0.3089 = -0.1768\)
\(\hat{y} = \sigma(\beta_1 - \theta_1) = \sigma(-0.1768 - 0.2) = \sigma(-0.3768) \approx 0.4069\)
\(E = \frac{1}{2}(\hat{y} - y)^2 = \frac{1}{2}(0.4069 - 0.3)^2 \approx 0.00571\)

(2) 梯度计算与更新

\(g_1 = \hat{y}(1-\hat{y})(y - \hat{y}) = 0.4069 \times 0.5931 \times (0.3 - 0.4069) \approx -0.0258\)

更新 \(w_{11}\)：\(\Delta w_{11} = \eta g_1 b_1 = 0.5 \times (-0.0258) \times 0.4403 \approx -0.0057\)
 \(w_{11} \leftarrow 0.3 + (-0.0057) = 0.2943\)

更新 \(w_{21}\)：\(\Delta w_{21} = \eta g_1 b_2 = 0.5 \times (-0.0258) \times 0.6177 \approx -0.0080\)
 \(w_{21} \leftarrow -0.5 + (-0.0080) = -0.5080\)

更新 \(\theta_1\)：\(\Delta\theta_1 = -\eta g_1 = -0.5 \times (-0.0258) = 0.0129\)
 \(\theta_1 \leftarrow 0.2 + 0.0129 = 0.2129\)

五、编程题 （20分）

13阅读以下使用 nnet 包训练神经网络的 R 代码，回答后面问题： 
library(nnet)
data(iris)
set.seed(123)
model <- nnet(Species ~ ., data = iris, 
              size = 5, maxit = 200, 
              decay = 0.01, MaxNWts = 2000)
pred <- predict(model, iris, type = "class")
conf_matrix <- table(iris$Species, pred)
cat("混淆矩阵:\n")
print(conf_matrix)
accuracy <- sum(diag(conf_matrix)) / sum(conf_matrix)
cat(sprintf("准确率: %.2f%%\n", accuracy * 100))

问题：

  1. 参数 size = 5 代表什么？它对应网络结构中的哪一部分？（4分）
  2. decay = 0.01 的作用是什么？在期末考试中常提到的"消除过拟合"方法中，这属于哪种方法？（3分）
  3. predict(model, iris, type = "class") 中 type = "class" 的含义是什么？如果不指定会怎样？（3分）

参考答案：

(1) size=5 表示隐藏层有5个神经元。这是单隐藏层网络结构中隐藏层的节点数。默认激活函数为 Sigmoid（logistic）。

(2) decay=0.01 是权重衰减（Weight Decay）正则化参数，等价于 L2 正则化。在期末考试的简答题中常考："神经网络中消除过拟合一般采用 L2 正则化方法"——nnet 中的 decay 参数正是 L2 正则化的实现。

(3) type="class" 返回预测的类别标签（如 setosa/versicolor）。如果不指定，默认 type="raw" 返回概率值矩阵。对于评估准确率等场景，需要显式指定 type="class"。
