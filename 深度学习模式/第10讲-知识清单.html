第10讲 知识清单：神经网络 — 感知机

2026年5月18日 · 上海对外经贸大学 · 王昭文 · 数据挖掘与机器学习
tag-must tag-key tag-freq tag-info

1. 直觉理解 Intuition 细读

人工神经网络 (ANN) 的核心思想：模仿生物神经元的信号处理机制——多个输入信号经加权求和后，与阈值比较，再经激活函数处理产生输出。

感知机 (Perceptron)：最简单的神经网络结构，由输入层 + 单层 M-P 神经元（输出层）组成，本质是在特征空间中寻找一个分离超平面来进行二分类。

类比理解：感知机就像在二维平面上画一条直线，把两类点分开。在高维空间中，这条"直线"变成了"超平面"。

直觉陷阱：感知机只能解决线性可分问题。如果数据线性不可分（如 XOR 问题），单层感知机永远无法收敛。

2. 核心定义 Definitions 细读

2.1 M-P 神经元模型 (McCulloch-Pitts, 1943) 细读

神经元：接收到来自 n 个其他神经元的输入信号，这些信号通过带权重的连接传递，神经元将接收到的总输入值与阈值比较，然后通过激活函数处理以产生输出。

$$y = f\left(\sum_{i=1}^{n} w_i x_i - \theta\right)$$

其中 w_i 为连接权值，θ 为阈值，f 为激活函数。

2.2 感知机模型 (Rosenblatt, 1958) 细读

感知机定义：输入空间 ⊆ ℝⁿ，输出空间 = {+1, -1}。由输入空间到输出空间的函数 f(x) = sign(w·x + b) 称为感知机，是一种线性分类模型，属于判别模型。

权值 w ℝⁿ 中的向量，决定超平面的方向（法向量）

偏置 b 实数，决定超平面的位置（截距）

分离超平面 线性方程 w·x + b = 0 对应于特征空间中的一个超平面 S

2.3 线性可分数据集 细读

线性可分：存在某个超平面 w·x + b = 0 能够将正实例点 (y=+1) 和负实例点 (y=-1) 完全正确地划分到超平面的两侧。

对所有 yᵢ = +1 的实例，有 w·xᵢ + b > 0；对所有 yᵢ = -1 的实例，有 w·xᵢ + b < 0。

3. 动机与背景 Motivation 细读

3.1 为什么需要感知机？ 细读

  • 神经网络的开山之作：1958年 Rosenblatt 提出，是第一个正式的人工神经网络模型，开启了连接主义 (Connectionism) 研究。
  • 可证伪的学习理论：对于线性可分数据，感知机算法被严格证明会在有限步内收敛。
  • 基础构建块：多层感知机 (MLP)) 和深度学习的基础——理解单层是理解多层的先决条件。

3.2 感知机的局限 细读

  • 异或问题 (XOR)：1969年 Minsky & Papert 证明单层感知机无法求解 XOR 问题，因为 XOR 是线性不可分的。
  • 第一次 AI 寒冬：这一缺陷导致神经网络研究停滞十余年。

4. 数学推导 Derivation 细读

4.1 感知机损失函数 细读

任一点 x₀ 到超平面 S 的距离：

$$\frac{|w \cdot x_0 + b|}{\|w\|}$$

其中 ‖w‖ 是 w 的 L₂ 范数。

对于误分类点 (xᵢ, yᵢ)：

当 w·xᵢ + b > 0 时，yᵢ = -1；当 w·xᵢ + b < 0 时，yᵢ = +1。

因此 -yᵢ(w·xᵢ + b) > 0 恒成立。误分类点 xᵢ 到超平面的距离为：

$$-\frac{1}{\|w\|} y_i (w \cdot x_i + b)$$

感知机损失函数（经验风险函数）：忽略 1/‖w‖，定义

$$L(w, b) = -\sum_{x_i \in M} y_i (w \cdot x_i + b)$$

M 为误分类点集合。该损失函数非负且关于 w, b 连续可导。本质是误分类点到超平面的总距离。

4.2 随机梯度下降 (随机梯度下降 (SGD)) 求解 细读

损失函数的梯度：

$$\nabla_w L(w, b) = -\sum_{x_i \in M} y_i x_i, \quad \nabla_b L(w, b) = -\sum_{x_i \in M} y_i$$

1算法输入/输出：

输入：训练集 T = {(x₁,y₁), …, (x_N, y_N)}，xᵢ ∈ ℝⁿ, yᵢ ∈ {-1, +1}，学习率 η (0 < η ≤ 1)

输出：w, b；感知机模型 f(x) = sign(w·x + b)

2迭代过程：

  1. 初始化 w₀, b₀
  2. 在训练集中随机选取数据 (xᵢ, yᵢ)
  3. 如果 yᵢ(w·xᵢ + b) ≤ 0（误分类条件），则更新：
    $$w \leftarrow w + \eta \cdot y_i \cdot x_i, \quad b \leftarrow b + \eta \cdot y_i$$
  4. 转至 (2)，直至训练集中没有误分类点。

几何直观：当一个实例被误分类时，调整 w, b 使分离超平面向该误分类点移动，直至超平面越过该点使其被正确分类。

4.3 收敛性定理 细读

Novikoff 定理：对于线性可分数据集，感知机算法在有限步内收敛（找到一个能够完全正确分开样本的超平面）。学习率 η 仅影响收敛速度，不影响最终收敛性。

记忆要点：收敛的充要条件是数据集线性可分。η 大小 → 不改变是否收敛，只改变迭代次数。

4.4 激活函数 细读

感知机使用 符号函数 (符号函数 (Sign) Function) 作为激活函数：

$$\text{sign}(z) = \begin{cases} +1, & z \geq 0 \\ -1, & z < 0 \end{cases}$$
激活函数公式取值范围特点用于
阶跃函数 (Step) 1 if z≥0 else 0 {0, 1} 不连续、不光滑 M-P 神经元基础
符号函数 (符号函数 (Sign)) ±1 {-1, +1} 不连续、不光滑 感知机
Sigmoid激活函数 1/(1+e⁻ᶻ) (0, 1) 光滑可导，导数最大 0.25，深层易梯度消失 多层感知机 (MLP) / 反向传播 算法
ReLU激活函数 max(0, z) [0, +∞) x>0 时导数为 1，缓解梯度消失 现代深度学习
Tanh激活函数 (eᶻ-e⁻ᶻ)/(eᶻ+e⁻ᶻ) (-1, 1) 零中心输出，比 Sigmoid激活函数 更优 RNN / LSTM

5. 实例演示 Examples 细读

5.1 感知机更新示例 细读

设有 3 个二维样本：正类 x₁=(3,3), y₁=+1；负类 x₂=(1,1), y₂=-1；初始 w=(0,0), b=0, η=1。

轮次误分类点w·x+b 值更新新 w新 b
1x₁ (正类)0 ≤ 0 ✘w←w+1·(+1)·(3,3), b←b+1(3,3)1
2x₂ (负类)3+3+1=7, y₂=-1 → -7 ≤ 0 ✘w←w+1·(-1)·(1,1), b←b-1(2,2)0
3x₁ (正类)6+6+0=12, y₁=+1 → 12 > 0 ✓正确，不更新——
4x₂ (负类)2+2+0=4, y₂=-1 → -4 ≤ 0 ✘w←(2,2)-1·(1,1), b←0-1(1,1)-1
5x₁ (正类)3+3-1=5, y₁=+1 → 5 > 0 ✓正确——
6x₂ (负类)1+1-1=1, y₂=-1 → -1 ≤ 0 ✘w←(1,1)-1·(1,1), b←-1-1(0,0)-2
7x₁ (正类)0+0-2=-2, y₁=+1 → -2 ≤ 0 ✘w←(0,0)+(3,3), b←-2+1(3,3)-1
8x₂ (负类)3+3-1=5, y₂=-1 → -5 ≤ 0 ✘w←(3,3)-1·(1,1), b←-1-1(2,2)-2

最终超平面：2x₁ + 2x₂ - 2 = 0，即 x₁ + x₂ - 1 = 0，将 (3,3) 和 (1,1) 正确分开。

5.2 课堂练习题精析 细读

Q1：感知机能够收敛的前提条件

答案：C — 数据集必须是线性可分的。感知机收敛的充要条件。A、B、D 均不是必要条件。

Q2：感知机损失函数 L(w,b) 在最小化什么

答案：B — 误分类点到超平面的总距离。不最小化错误个数（不可导），不是距离平方和，不是交叉熵。

Q3：权重更新的几何解释

答案：B — 让超平面向该误分类点旋转，试图将其包含在正确的一侧。w ← w + ηyᵢxᵢ 使法向量向正确方向偏转。

Q4：感知机的激活函数

答案：C — 符号函数 (符号函数 (Sign) Function)。Sigmoid激活函数、ReLU激活函数、Tanh激活函数 是 多层感知机 (MLP) 中使用的激活函数。

Q5：学习率 η 对收敛的影响

答案：C — 只要数据线性可分，η 的大小不影响收敛性（仅影响迭代次数）。η 控制步长，步长大可能导致"过冲"，但不改变能否收敛。

7. 常见陷阱与注意点 Pitfalls 细读

陷阱1：误以为感知机能处理任意分类问题。
单层感知机只能处理线性可分的数据集。XOR 问题、同心圆等非线性问题无法处理。
陷阱2：混淆损失函数和误分类个数。
感知机的损失函数是最小化误分类点到超平面的总距离，而非误分类样本个数。因为误分类个数不可导，无法用梯度下降优化。
陷阱3：忘记收敛的前提条件。
感知机算法 仅在数据线性可分时收敛。若数据线性不可分，算法将无限循环（不收敛），需要在训练前做停止条件限制。
陷阱4：以为学习率 η 影响收敛性。
η 只影响收敛速度和迭代次数，不影响是否收敛。即使 η 很大导致震荡，只要数据线性可分，最终仍会收敛。注意 η ∈ (0, 1]。
陷阱5：将感知机与逻辑回归混淆。
感知机使用 sign 函数输出 ±1，属于硬分类；逻辑回归使用 sigmoid 输出概率，属于软分类。两者虽都是线性模型，但损失函数和输出完全不同。
陷阱6：以为线性激活函数叠加多层有意义。
如果在 多层感知机 (MLP) 所有层使用线性激活函数，整个网络等价于一个单层线性模型——多层无用！必须使用非线性激活函数才能获得深层网络的优势。

8. 多层感知机 (MLP) 与反向传播扩展知识 细读

8.1 万能近似定理 (Universal Approximation Theorem) 细读

核心结论：只要神经元数量足够多，仅需一个隐藏层的神经网络就能以任意精度近似任何连续函数。

  • 隐藏层数量：理论上仅需 1 层（宽度足够）
  • 激活函数 σ：必须是非多项式的（线性的不行）
  • 紧集 K：函数必须在一个有限范围内进行逼近
深层 vs 宽层：虽然单隐藏层理论上可以拟合任何函数，但可能需要指数级数量的神经元（维数灾难）。深层网络在表达具有层次结构的数据时比单层网络高效得多。

8.2 反向传播 (Back Propagation, BP) 算法 细读

前向传播：输入信号通过各层神经元处理，最终在输出层产生预测值，并计算与真实值的损失 (Loss)。

反向传播：将损失从输出层开始，沿着网络结构"反向"传播，通过链式法则 (Chain Rule) 计算每一层权重对损失的贡献（即梯度），进而更新权重。

$$\frac{\partial E_k}{\partial w_{hj}} = \frac{\partial E_k}{\partial \hat{y}_j^k} \cdot \frac{\partial \hat{y}_j^k}{\partial \beta_j} \cdot \frac{\partial \beta_j}{\partial w_{hj}}$$

链式法则：误差对权重的偏导 = 误差对输出的偏导 × 输出对输入的偏导 × 输入对权重的偏导。

8.3 梯度消失 (Vanishing Gradient) 细读

原因：传统 Sigmoid 函数的导数最大值仅 0.25。在深层网络中，连乘项 0.25ⁿ 迅速衰减至接近 0，导致靠近输入层的权重几乎不更新。

解决方案：使用 ReLU激活函数（x>0 时导数为 1）替代 Sigmoid激活函数，有效缓解梯度消失问题。

8.4 Mini-batch 的优势 细读

  • 统计意义：总体梯度的无偏估计；Batch Size 增大，梯度方差降低
  • 计算优势：利用 GPU 并行性，处理 Batch 的时间远比串行处理快
  • 噪声作为正则化：适度噪声帮助跳出局部最小值，找到泛化更好的平坦区域
