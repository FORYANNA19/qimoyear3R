第11讲：神经网络 — BP反向传播算法

上海对外经贸大学 · 数据挖掘与机器学习（王昭文）  |  更新：2026年5月25日

1多层感知机 (MLP) 多层感知机架构 细读

Step 1: 理解神经网络的基本拓扑结构
必考 核心概念

多层前馈神经网络 (Multi-Layer Feedforward Neural Network) 是深度学习的基础架构。 每层神经元与下一层神经元全互连 (fully connected)， 但不存在同层连接，也不存在跨层连接。

【网络拓扑图】
Input Layer (d) ────→ Hidden Layer (q) ────→ Output Layer (l)
x₁, x₂, ..., xd   →   b₁, b₂, ..., bq   →   ŷ₁, ŷ₂, ..., ŷl
单层感知机 vs 多层感知机：多层感知机 (MLP) 引入了隐藏层 (Hidden Layer)， 从而能够处理线性不可分问题。如果 多层感知机 (MLP) 中全部使用线性激活函数， 则整个网络等价于一个简单的线性模型（多层线性变换的复合依然是线性变换）。
符号含义维度/范围
d输入层神经元个数= 输入属性维度
q隐层神经元个数超参数
l输出层神经元个数= 输出维度
vih输入层 i → 隐层 h 的连接权d × q 个参数
whj隐层 h → 输出层 j 的连接权q × l 个参数
γh隐层第 h 个神经元的阈值 (bias)q 个
θj输出层第 j 个神经元的阈值 (bias)l 个

可训练参数总数（不计偏置）：d × q + q × l
例如：d=10, q=20, l=5 → 10×20 + 20×5 = 300 个权重参数

2前向传播 (Forward Propagation) 细读

Step 2: 输入信号流经网络，生成预测并计算损失
必考 核心流程

训练集 \(D = \{(\mathbf{x}_1, \mathbf{y}_1), (\mathbf{x}_2, \mathbf{y}_2), \dots, (\mathbf{x}_m, \mathbf{y}_m)\}\), 其中 \(\mathbf{x}_i \in \mathbb{R}^d\), \(\mathbf{y}_i \in \mathbb{R}^l\).

隐层第 h 个神经元的输入（加权和）： $$\alpha_h = \sum_{i=1}^{d} v_{ih} \, x_i$$ 隐层第 h 个神经元的输出（经激活函数）： $$b_h = f(\alpha_h - \gamma_h)$$ 输出层第 j 个神经元的输入： $$\beta_j = \sum_{h=1}^{q} w_{hj} \, b_h$$ 输出层第 j 个神经元的预测值： $$\hat{y}_j = f(\beta_j - \theta_j)$$
其中 \(f(\cdot)\) 为激活函数，通常使用 Sigmoid激活函数：\(f(x) = \frac{1}{1+e^{-x}}\)， 满足性质 \(f'(x) = f(x)(1 - f(x))\)。

前向传播的目的：给定输入 \(\mathbf{x}\)，逐层计算直至输出 \(\hat{\mathbf{y}}\)， 然后与真实标签 \(\mathbf{y}\) 比较，计算损失 \(E_k\)（通常为均方误差）： $$E_k = \frac{1}{2} \sum_{j=1}^{l} (\hat{y}_j^k - y_j^k)^2$$

3反向传播 推导：链式法则 (Chain Rule) 细读

Step 3: 损失沿网络反向传播，逐层求梯度
必考 核心推导 高频

反向传播 算法基于梯度下降策略，以目标函数的负梯度方向对参数进行调整。 更新权重的核心数学工具是微积分中的链式法则 (Chain Rule)。

▎ \(w_{hj}\) 的梯度链式分解：

\(w_{hj}\) 先影响输出层神经元 j 的输入值 \(\beta_j\)，再影响其输出 \(\hat{y}_j^k\)，最后影响误差 \(E_k\)：

$$\frac{\partial E_k}{\partial w_{hj}} = \frac{\partial E_k}{\partial \hat{y}_j^k} \cdot \frac{\partial \hat{y}_j^k}{\partial \beta_j} \cdot \frac{\partial \beta_j}{\partial w_{hj}}$$
三项分别计算：
① \(\displaystyle \frac{\partial \beta_j}{\partial w_{hj}} = b_h\)（由定义 \(\beta_j = \sum_h w_{hj} b_h\) 直接得出）
② \(\displaystyle \frac{\partial \hat{y}_j^k}{\partial \beta_j} = f'(\beta_j - \theta_j)\)（Sigmoid激活函数 导数）
③ \(\displaystyle \frac{\partial E_k}{\partial \hat{y}_j^k} = \hat{y}_j^k - y_j^k\)（均方误差 (MSE) 导数，负号已吸收进 g_j 定义）
▎ 定义梯度中间变量 \(g_j\)（输出层误差信号）： $$g_j = -\frac{\partial E_k}{\partial \hat{y}_j^k} \cdot \frac{\partial \hat{y}_j^k}{\partial \beta_j} = (\hat{y}_j^k - y_j^k) \cdot f'(\beta_j - \theta_j)$$
对于 Sigmoid 函数，利用 \(f'(x)=f(x)(1-f(x))\) 可得简化形式： $$g_j = \hat{y}_j^k (1 - \hat{y}_j^k)(y_j^k - \hat{y}_j^k)$$ 这是最常用的实用形式，只需知道预测值 \(\hat{y}_j^k\) 和真实值 \(y_j^k\) 即可计算。
▎ 隐层误差信号 \(e_h\)（反向传播至隐层）： $$e_h = -\frac{\partial E_k}{\partial b_h} \cdot \frac{\partial b_h}{\partial \alpha_h} = \sum_{j=1}^{l} w_{hj}\, g_j \cdot f'(\alpha_h - \gamma_h)$$
对于 Sigmoid激活函数 隐层神经元： $$e_h = b_h (1 - b_h) \sum_{j=1}^{l} w_{hj} \, g_j$$ \(e_h\) 综合了所有输出层误差 \(g_j\) 经权重 \(w_{hj}\) 反向传播至隐层的结果。
混淆 \(g_j\)（输出层误差）和 \(e_h\)（隐层误差）的公式。\(g_j\) 只需要局部信息（\(\hat{y}_j^k, y_j^k\)）， \(e_h\) 需要聚合所有输出层的 \(g_j\) 乘以对应权重 \(w_{hj}\)。这是 反向传播 最核心的两步反向传递。

4梯度计算公式汇总

Step 4: 四个核心参数的梯度公式
必背 公式卡
参数梯度公式更新公式 \(\Delta\)
whj（隐→输出权重） \(\displaystyle \frac{\partial E_k}{\partial w_{hj}} = -g_j \, b_h\) \(\Delta w_{hj} = \eta \, g_j \, b_h\)
θj（输出层阈值） \(\displaystyle \frac{\partial E_k}{\partial \theta_j} = g_j\) \(\Delta \theta_j = -\eta \, g_j\)
vih（输入→隐层权重） \(\displaystyle \frac{\partial E_k}{\partial v_{ih}} = -e_h \, x_i\) \(\Delta v_{ih} = \eta \, e_h \, x_i\)
γh（隐层阈值） \(\displaystyle \frac{\partial E_k}{\partial \gamma_h} = e_h\) \(\Delta \gamma_h = -\eta \, e_h\)
统一规律：权重更新 = 学习率 × 误差信号 × 输入信号； 阈值更新 = −学习率 × 误差信号。 权重更新中始终包含"上一层神经元的输出"作为因子。

5权重更新与学习率 细读

Step 5: 梯度下降的参数更新策略
关键 常考

学习率 \(\eta\) 控制算法每一轮迭代中的更新步长：

  • \(\eta\) 太大：容易振荡 (oscillation)，在最优解附近反复横跳，无法收敛
  • \(\eta\) 太小：收敛速度过慢，训练时间过长
  • \(\eta\) 适中：平衡收敛速度与稳定性，现代常用 Adam 等自适应学习率方法

标准 反向传播 算法的参数更新（对单个样本 \((\mathbf{x}_k, \mathbf{y}_k)\)）：

$$w_{hj} \leftarrow w_{hj} + \eta \, g_j \, b_h$$ $$\theta_j \leftarrow \theta_j - \eta \, g_j$$ $$v_{ih} \leftarrow v_{ih} + \eta \, e_h \, x_i$$ $$\gamma_h \leftarrow \gamma_h - \eta \, e_h$$

6Mini-Batch 与优化器对比 不考

Step 6: 批量策略与优化算法的选择
必考 高频概念
策略每次梯度计算样本数优点缺点
全批量 (Full Batch) 全部 m 个样本 梯度估计准确，收敛稳定 内存吃不消，计算极慢
随机梯度下降 (随机梯度下降 (SGD)) 1 个样本 计算快，可在线更新 梯度估计方差极大，路径"横冲直撞"
Mini-Batch 小批量（32/64/128） GPU 并行高效；梯度方差适中；噪声助正则化 需调 batch size 超参

Mini-Batch 的深层原理：

  • 统计学意义：是总体梯度的无偏估计；根据大数定律，batch size 越大，梯度方差越低
  • 计算优势：现代 GPU 高度并行，处理一个 batch 的时间不比处理单个样本慢多少，极大提高吞吐量
  • 噪声正则化：Mini-batch 带来的适度噪声能帮助模型跳出局部最小值 (shallow pit)，找到泛化能力更好的平坦区域

7反向传播 算法完整流程总结 细读

Step 7: 端到端的训练流程与常见陷阱
必考 综述

反向传播 算法一次迭代的完整步骤：

  1. 前向传播：输入 \(\mathbf{x}_k\) → 计算 \(\alpha_h, b_h\)（隐层）→ 计算 \(\beta_j, \hat{y}_j\)（输出层）→ 算损失 \(E_k\)
  2. 反向传播输出层：计算 \(g_j = \hat{y}_j(1-\hat{y}_j)(y_j - \hat{y}_j)\) → 更新 \(w_{hj}, \theta_j\)
  3. 反向传播隐层：计算 \(e_h = b_h(1-b_h)\sum_j w_{hj} g_j\) → 更新 \(v_{ih}, \gamma_h\)
  4. 参数更新：按梯度下降公式更新所有权重和阈值
  5. 重复：对下一个 mini-batch 重复以上步骤，直至收敛
常见错误：在反向传播阶段计算的是"损失函数对网络中各个权值和偏置的梯度"， 而不是激活值、特征分布或预测准确率。前向传播计算激活值和预测值，反向传播只计算梯度。
激活函数选择：如果在 多层感知机 (MLP) 中使用线性激活函数 \(f(x)=x\)，由于多层线性变换的复合依然是线性变换， 整个 多层感知机 (MLP) 等价于一个简单的线性模型，失去了引入非线性（处理线性不可分问题）的根本意义。
批量 (Batch) 概念：教材中的 Batch 指"在更新一次参数之前，计算多个样本的平均梯度"， 而不是一次性处理整个生命周期的数据，也不是隐藏层的层数。
