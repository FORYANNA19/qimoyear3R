三、什么是机器学习 必考

3.1 AI → ML → DL 的关系

  • 人工智能（AI）重点：由计算机科学、控制论、信息论、神经生理学、心理学、数学、统计学、哲学等交叉形成的综合性学科

3.3 Mitchell 的机器学习定义 必考

Mitchell (1997) 定义：

中文：假定用P来评估计算机程序在某个任务T上的性能，若一个程序通过利用经验E在T上的任务获得了改善，则我们说关于T和P，该程序对E进行了学习。
英文：A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.
  • 任务T (Task)：任务 —— 要完成的事情（如：下围棋、预测天气）
  • 性能目标P (Performance)：性能目标 —— 评价做得好不好（如：击败对手的百分比）
  • 训练经验E (Experience)：经验 —— 学习的来源（如：与自己对弈、看棋谱）

实例：围棋

  • T：下围棋
  • P：比赛中击败对手的百分比
  • E：和自己进行对弈，或看棋谱
很多人混淆 P（性能度量）和 T（任务本身），例如把"预测天气"当作P，而P应该是"预测正确的概率"。

3.4 实现机器学习的五个步骤

机器学习研究的对象是数据，即从数据出发，提取数据的特征或者规律，抽象出数据模型，利用学习的模型对新数据进行预测或分析

  1. 得到一个有限的训练数据集合
  2. 确定包含所有可能模型的假设空间（即学习模型的集合）
  3. 确定模型选择的准则，即学习的策略
  4. 通过某种算法求解最优的模型
  5. 利用学习的最优模型对新数据进行预测或分析
机器学习 = 数据 + 模型 + 策略 + 算法

六、机器学习的分类 必考

本节概要：机器学习按学习方式分为监督学习、无监督学习、半监督学习和强化学习，必须掌握各类型的特征与区别。

6.1 三大类别 必考

  • 监督学习（Supervised Learning）：有输入(X)也有输出(Y)标签数据，学习从X到Y的映射
  • 无监督学习（Unsupervised Learning）：只有输入(X)没有输出标签，任务是描述数据如何组织或聚集
  • 强化学习（Reinforcement Learning）：通过与环境交互获得奖励/惩罚来学习策略

6.2 监督学习 必考

定义

在典型的场景中，我们感兴趣的输出结果通常是定量的（如股票价格）或分类的（如购买/不购买），我们根据一组输入/特征来预测输出。

基本设定

  • 输入变量X：又称特征向量(feature vector)，输入变量取值的空间称为特征空间
  • 输出变量Y：输出变量取值的空间称为输出空间
  • 训练数据集：$$\mathcal{T} = \{(\mathbf{x}_1, y_1), (\mathbf{x}_2, y_2), \ldots, (\mathbf{x}_N, y_N)\}$$ 共N个样本，每个样本包含输入x_i和对应输出y_i
  • 决策函数：目标是从训练数据中学习一个映射 $$f: \mathcal{X} \to \mathcal{Y}$$，使得对新的输入 $$x_{new}$$，有预测值 $$\hat{y} = f(x_{new})$$

两类问题

  • 分类（Classification）：输出Y是离散类别标签，如垃圾邮件 vs 正常邮件，数字0-9
  • 回归（Regression）：输出Y是连续数值，如股票价格、股价交易量

常见监督学习算法

  • 线性回归 (Linear Regression)
  • 逻辑回归 (Logistic Regression)
  • 贝叶斯分类 (Naive Bayes)
  • 决策树 (Decision Tree)
  • 支持向量机)
  • K近邻法 (K-Nearest Neighbors)
  • 神经网络 (Neural Network)
分类和回归的选择依赖于输出变量的类型，不是输入变量的类型。预测"明天是否下雨"是分类问题（Yes/No），预测"明天的降雨量"是回归问题。预测"明天微软股票交易量"是回归问题，因为交易量是连续数值。

6.3 无监督学习 重点

定义

在无监督学习中，我们只观察特征向量而没有测量输出。任务是描述数据是如何组织或聚集的。

训练集形式

$$\mathcal{T} = \{\mathbf{x}_1, \mathbf{x}_2, \ldots, \mathbf{x}_N\}, \quad \mathbf{x}_i \in \mathbb{R}^p$$ 只有输入特征，没有输出标签。

主要类型

  • 聚类（Clustering）：找到这样的簇 —— 同一族内数据点更"相似"，不同族之间"不太相似"
    • 市场分割：根据客户变量找到相似客户群
    • 文档聚类：根据重要术语找出相似文档
    • 股票分类：根据每日走势进行分类
    • DNA微阵列分析 / 谷歌新闻专题聚类
  • 降维（Dimensionality Reduction）：减少特征数量，同时保留主要信息

6.4 半监督学习（Semi-supervised Learning）

同时利用有标记数据(labelled)和无标记数据(unlabelled)来建立分类或预测模型。数据：$$\mathcal{T} = \{(\mathbf{x}_1, y_1), \ldots, (\mathbf{x}_l, y_l)\} \cup \{\mathbf{x}_{l+1}, \ldots, \mathbf{x}_N\}$$，通常$$l \ll N$$（标记数据远少于未标记数据）。应用：识别网页真假新闻、医学影像分析、网页推荐。

6.5 没有免费的午餐（No Free Lunch Theorem）重点

核心思想：如果算法A在情况1下表现优于算法B，那么必然存在情况2使得算法B优于算法A —— 理论上没有一致最优的算法！

脱离实际情况而空谈哪个算法好是没有意义的。

七、机器学习三要素 必考

本节概要：任何机器学习方法都由模型、策略、算法三要素构成，是课程最核心的理论框架。

7.1 模型（Model）重点

机器学习首先要考虑的是学习什么样的模型。在监督学习中，模型就是要学习的决策函数或条件概率分布。

假设空间：$$\mathcal{F} = \{f \mid Y = f(X)\} \quad \text{或} \quad \mathcal{F} = \{P \mid P(Y \mid X)\}$$ 假设空间包含所有可能的条件概率分布或决策函数的集合。

7.2 策略（Strategy）必考

策略回答的问题是：按照什么样的准则从假设空间中选择最优模型。

损失函数（Loss Function）

定义：损失函数用来度量预测错误的程度，是预测值 $$f(X)$$ 与真实值 $$Y$$ 的非负实值函数，记作 $$L(Y, f(X))$$。

名称公式说明
0-1损失 (0-1 Loss) $$L(Y, f(X)) = \begin{cases} 0, & Y = f(X) \\ 1, & Y \neq f(X) \end{cases}$$ 分类问题最基本损失
平方损失 (Squared Loss) $$L(Y, f(X)) = (Y - f(X))^2$$ 回归问题最常用
绝对值损失 (Absolute Loss) $$L(Y, f(X)) = |Y - f(X)|$$ 对异常值更稳健
对数损失 (Log Loss) $$L(Y, P(Y \mid X)) = -\log P(Y \mid X)$$ 分类概率模型
Huber损失 L2和L1的组合 综合两者优点
合页损失 (Hinge Loss) $$L(Y, f(X)) = \max(0, 1 - Y \cdot f(X))$$ SVM使用

期望损失（风险函数）

$$R_{\text{exp}}(f) = \mathbb{E}_{(X,Y) \sim P}[L(Y, f(X))] = \int L(y, f(x)) \, dP(x, y)$$ 理论上决策函数 $$f(X)$$ 关于联合分布 $$P(X,Y)$$ 的平均损失，又称风险函数 (Risk Function) 或期望损失 (Expected Loss)。

理论上最优决策函数：$$f^* = \arg\min_{f \in \mathcal{F}} R_{\text{exp}}(f)$$

经验损失（经验风险）必考

由于 $$P(X,Y)$$ 未知，用训练数据上的平均损失来代替：

$$R_{\text{emp}}(f) = \frac{1}{N} \sum_{i=1}^{N} L(y_i, f(\mathbf{x}_i))$$ 经验风险最小化（ERM）： $$f^* = \arg\min_{f \in \mathcal{F}} R_{\text{emp}}(f)$$
当样本量很小时，经验风险最小化容易产生"过拟合"（over-fitting）！因为模型可能过度拟合训练数据的噪声，导致泛化能力差。

结构风险最小化（SRM）必考

为防止过拟合，在经验风险上加入正则化项（regularizer）/罚项（penalty term）：

$$R_{\text{str}}(f) = R_{\text{emp}}(f) + \lambda \cdot J(f)$$ 其中：
  • $$J(f)$$：模型复杂度，模型越复杂 $$J(f)$$ 越大
  • $$\lambda > 0$$：正则化系数，权衡经验风险和模型复杂度
结构风险最小化： $$f^* = \arg\min_{f \in \mathcal{F}} \left[ \frac{1}{N} \sum_{i=1}^{N} L(y_i, f(\mathbf{x}_i)) + \lambda \cdot J(f) \right]$$

本质：结构风险最小化 = 经验风险最小化 + 正则化（Regularization），在拟合数据和模型复杂度之间取得平衡。

7.3 算法（Algorithm）重点

算法是指学习模型的具体计算方法。机器学习问题最终都会归结为最优化问题。

机器学习方法的不同主要来自于模型、策略以及算法的不同。

八、线性回归模型（三要素实例）必考

本节概要：以线性回归为例，完整展示ML三要素如何具体落地。

8.1 模型

决策函数：$$f(\mathbf{x}) = \mathbf{w}^T \mathbf{x} + b$$ 或写成：$$f(\mathbf{x}) = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p$$ 假设空间：$$\mathcal{F} = \{f \mid f(\mathbf{x}) = \mathbf{w}^T \mathbf{x} + b, \mathbf{w} \in \mathbb{R}^p, b \in \mathbb{R}\}$$

8.2 策略

采用L-2损失（平方损失），通过最小二乘法求解：

$$\min_{\beta} \sum_{i=1}^{N} (y_i - \beta_0 - \beta_1 x_{i1} - \cdots - \beta_p x_{ip})^2$$ 写成矩阵形式（含截距项）： $$\min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2$$ 其中 $$\mathbf{X}$$ 是 $$N \times (p+1)$$ 设计矩阵，第一列全为1（截距项）。

8.3 算法

解析解（正规方程 Normal Equation）：

$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$ 可通过 QR分解 或 Cholesky分解 求解最小二乘问题。

8.4 梯度下降法 必考

当数据量大或 $$X^TX$$ 不可逆时，使用迭代优化方法：

目标函数：$$J(\boldsymbol{\beta}) = \frac{1}{N} \sum_{i=1}^{N} (y_i - \mathbf{x}_i^T \boldsymbol{\beta})^2$$ 梯度：$$\nabla J(\boldsymbol{\beta}) = -\frac{2}{N} \mathbf{X}^T (\mathbf{y} - \mathbf{X}\boldsymbol{\beta})$$ 迭代公式：$$\boldsymbol{\beta}^{(k+1)} = \boldsymbol{\beta}^{(k)} - \eta \cdot \nabla J(\boldsymbol{\beta}^{(k)})$$ 其中 $$\eta$$ 为步长/学习率(learning rate)。

梯度下降法步骤

  1. 给定初始值 $$\boldsymbol{\beta}^{(0)}$$ 以及步长 $$\eta$$
  2. 更新：$$\boldsymbol{\beta}^{(k+1)} = \boldsymbol{\beta}^{(k)} - \eta \cdot \nabla J(\boldsymbol{\beta}^{(k)})$$
  3. 重复步骤2，直到满足收敛标准

8.5 结构风险：岭回归（Ridge Regression）重点

在线性回归中加入L-2正则化：

$$\hat{\boldsymbol{\beta}}_{\text{ridge}} = \arg\min_{\boldsymbol{\beta}} \left[ \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2 + \lambda \|\boldsymbol{\beta}\|_2^2 \right]$$ 解析解：$$\hat{\boldsymbol{\beta}}_{\text{ridge}} = (\mathbf{X}^T \mathbf{X} + \lambda \mathbf{I})^{-1} \mathbf{X}^T \mathbf{y}$$

九、逻辑回归（Logistic Regression）必考

本节概要：逻辑回归是分类问题的基础模型，通过Sigmoid函数将线性输出映射为概率。

9.1 模型

核心思想：构造输入X与输出Y（取0/1值）的条件概率模型 $$P(Y \mid X)$$。

Sigmoid函数 必考

$$P(Y = 1 \mid \mathbf{x}) = \frac{1}{1 + e^{-(\mathbf{w}^T \mathbf{x} + b)}} = \sigma(\mathbf{w}^T \mathbf{x} + b)$$ 其中 $$\sigma(z) = \frac{1}{1 + e^{-z}}$$ 为 Sigmoid 函数。

设计原则：① 将线性输出映射到[0,1]区间（作为概率）；② 线性输出越大，变换后的概率越大。

9.2 对数几率（Logit）重点

几率（Odds）：$$\text{odds} = \frac{P(Y=1)}{P(Y=0)} = \frac{p}{1-p}$$ 对数几率（Logit）：$$\log\frac{p}{1-p} = \mathbf{w}^T \mathbf{x} + b$$

这解释了名称由来：Logistic回归实际上是在用线性回归模型的预测结果去逼近真实标记的对数几率，因此又被称为"对数几率回归"。

9.3 参数求解

似然函数

记 $$p_i = P(Y = 1 \mid \mathbf{x}_i) = \sigma(\mathbf{w}^T \mathbf{x}_i + b)$$，则：

似然函数：$$L(\mathbf{w}) = \prod_{i=1}^{N} p_i^{y_i} (1 - p_i)^{1 - y_i}$$ 负对数似然（损失函数）： $$\ell(\mathbf{w}) = -\sum_{i=1}^{N} \left[ y_i \log p_i + (1 - y_i) \log (1 - p_i) \right]$$

逻辑回归的优化问题：

$$\min_{\mathbf{w}} \ell(\mathbf{w}) = \min_{\mathbf{w}} -\sum_{i=1}^{N} \left[ y_i \log \sigma(\mathbf{w}^T \mathbf{x}_i) + (1 - y_i) \log (1 - \sigma(\mathbf{w}^T \mathbf{x}_i)) \right]$$ 注：没有显式解析解！需要用迭代优化方法。

梯度下降法求解

梯度：$$\nabla \ell(\mathbf{w}) = -\sum_{i=1}^{N} (y_i - p_i) \mathbf{x}_i = -\mathbf{X}^T (\mathbf{y} - \mathbf{p})$$ 迭代公式：$$\mathbf{w}^{(k+1)} = \mathbf{w}^{(k)} - \eta \cdot \nabla \ell(\mathbf{w}^{(k)})$$

9.4 分类决策 重点

学到模型后，对新输入x的分类决策：

$$\hat{y} = \begin{cases} 1, & \text{if } P(Y=1 \mid \mathbf{x}) \geq 0.5 \\ 0, & \text{otherwise} \end{cases}$$ 等价于：$$\hat{y} = 1 \iff \mathbf{w}^T \mathbf{x} + b \geq 0$$

结论：本质上，逻辑回归是线性分类器！决策边界为 $$\mathbf{w}^T \mathbf{x} + b = 0$$。

很多人误以为逻辑回归是非线性分类器，因为它用了Sigmoid变换。实际上Sigmoid只是将线性输出映射到概率，决策边界 $$\mathbf{w}^T \mathbf{x} + b = 0$$ 仍然是一条直线（超平面）。

十、本讲小结

  • 机器学习定义（Mitchell 1997）：T（任务）、P（性能度量）、E（经验）
  • 机器学习分类：监督学习 vs 无监督学习 vs 强化学习
  • 监督学习 vs 无监督学习的核心区别：是否有输出标签Y
  • 分类 vs 回归：输出Y是离散还是连续
  • 机器学习三要素：模型、策略、算法
  • 损失函数：0-1损失、平方损失、对数损失等
  • 经验风险 vs 期望风险 vs 结构风险
  • 结构风险 = 经验风险 + λ·模型复杂度（正则化）
  • 线性回归：模型=线性函数，策略=最小二乘(L2损失)，算法=正规方程/梯度下降
  • 逻辑回归：Sigmoid函数 → 对数几率 → 负对数似然 → 梯度下降 → 线性分类器
  • No Free Lunch Theorem：没有一致最优的算法
