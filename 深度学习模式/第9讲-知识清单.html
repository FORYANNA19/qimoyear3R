5. 直觉理解 — 神经网络入门

神经网络由具有适应性的简单单元组成的广泛并行互连的网络，它的组织能够模拟生物神经系统对真实世界物体所作出的交互反应。（T. Kohonen, 1988）

核心直觉：模仿生物神经元——接收信号 → 累加加权求和 → 与阈值比较 → 激活函数产生输出。多个神经元分层连接构成网络。

关键启示：单层感知机只能解决线性可分问题。要解决 XOR 等非线性问题，必须引入隐藏层，形成多层感知机 (MLP))。

6. 核心定义 — 神经元与感知机 细读

6.1 M-P 神经元模型

M-P神经元模型 (McCulloch & Pitts, 1943)：接收 n 个输入信号，通过带权重的连接传递，总输入与阈值比较后，经激活函数处理产生输出。

$$y = f\left(\sum_{i=1}^{n} w_i x_i - \theta\right)$$

xᵢ：第 i 个输入信号，wᵢ：第 i 个连接的权重，θ：阈值，f：激活函数

6.2 感知机模型 (Perceptron)

Frank Rosenblatt 于 1958 年提出，是 ANN 的第一个正式模型。

感知机属于判别模型，是一种线性分类模型。

$$f(x) = \text{sign}(w \cdot x + b)$$

w ∈ Rⁿ：权值向量（超平面的法向量），b ∈ R：偏置（超平面的截距）

分离超平面：w·x + b = 0，将特征空间分为正负两类

激活函数：符号函数 sign(·)，输出 +1 或 -1

6.3 线性可分性

线性可分数据集：存在超平面 w·x + b = 0，使得所有正实例点满足 w·x + b > 0，所有负实例点满足 w·x + b < 0。

感知机只能处理线性可分问题。XOR 问题线性不可分，导致 AI 第一次寒冬。

7. 核心定义 — 感知机学习策略 细读

7.1 损失函数

感知机的损失函数：误分类点到超平面 S 的总距离（不是误分类样本个数）

$$L(w, b) = -\sum_{x_i \in M} y_i (w \cdot x_i + b)$$

M：误分类点集合。对误分类点 (xᵢ, yᵢ)，-yᵢ(w·xᵢ + b) > 0 恒成立。

性质：L(w,b) 非负且关于 w,b 连续可导。

优化目标：在假设空间中选取使 L(w,b) 最小的模型参数 w,b。

7.2 损失函数的来源

点 x₀ 到超平面 w·x + b = 0 的距离（不含 1/||w||）：

$$\frac{1}{\|w\|}|w \cdot x_0 + b|$$

对误分类点，yᵢ 与 w·xᵢ + b 符号相反，所以：

$$|w \cdot x_i + b| = -y_i (w \cdot x_i + b)$$

8. 数学推导 — 感知机学习算法 细读

8.1 随机梯度下降法 (随机梯度下降 (SGD))

假设误分类点集合 M 固定，损失函数 L(w,b) 的梯度：

$$\nabla_w L(w, b) = -\sum_{x_i \in M} y_i x_i$$ $$\nabla_b L(w, b) = -\sum_{x_i \in M} y_i$$

随机选取一个误分类点 (xᵢ, yᵢ)，对 w, b 进行更新：

$$w \leftarrow w + \eta y_i x_i$$ $$b \leftarrow b + \eta y_i$$

η (eta)：学习率，0 < η ≤ 1。

8.2 感知机算法原始形式

1选取初值 w₀, b₀

2在训练集中选取数据 (xᵢ, yᵢ)

3如果 yᵢ(w·xᵢ + b) ≤ 0（误分类），则更新：w ← w + ηyᵢxᵢ，b ← b + ηyᵢ

4转至第 2 步，直至训练集中没有误分类点

几何直观：当一个实例点被误分类时，调整 w, b 使超平面向该误分类点的一侧移动，以减少该点与超平面间的距离，直到超平面越过该点使其被正确分类。

8.3 梯度下降法一般形式

对于无约束优化问题 min f(x)：

$$x^{(k+1)} \leftarrow x^{(k)} - \lambda_k \nabla f(x^{(k)})$$

一阶泰勒展开：\(f(x) = f(x^{(k)}) + g_k^T (x - x^{(k)})\)，其中 gk = ∇f(x(k))

当目标函数是凸函数时，梯度下降法收敛到全局最优解。

8.4 对偶形式

基本想法：将 w 和 b 表示为实例 xᵢ 和标记 yᵢ 的线性组合形式，通过求系数而求得 w 和 b。

经多次迭代后：

$$w = \sum_{i=1}^{N} \alpha_i y_i x_i, \quad b = \sum_{i=1}^{N} \alpha_i y_i$$

主要优势：当样本维度 D 远大于样本数 N 时，可通过预计算 Gram 矩阵加速训练。

决策函数完全由输入向量之间的内积表示。

9. 深刻洞见 — 感知机收敛与局限性 细读

9.1 收敛定理 (Novikoff)

对于线性可分数据集，感知机算法经有限次迭代可收敛。

令 \(R = \max_i \|\hat{x}_i\|\)，γ 为最小间隔（\(y_i(\hat{w}_{opt} \cdot \hat{x}_i) \ge \gamma\)），则误分类次数 k 满足：

$$k \le \left(\frac{R}{\gamma}\right)^2$$

9.2 重要性质

性质说明
解不唯一不同初值或不同误分类点选取顺序 → 不同解
学习率不影响收敛性只要数据线性可分，η > 0 均收敛（仅影响迭代次数）
不能处理 XOR单层感知机无法解决非线性可分问题 → AI 第一次寒冬
批量学习 vs 在线学习批量：收集全部数据再更新；在线：来一个样本更新一次
深刻洞见：感知机与 支持向量机 的核心连接——两者都寻找分离超平面，但感知机只要求「分开」，支持向量机 要求「最大化间隔」。支持向量机 正是从感知机的局限性（解不唯一、对离群点敏感）中找到的改进方向。

12. 多层感知机 (MLP)) 入门 细读

12.1 从单层到多层

XOR 问题：单层感知机无法解决 XOR → 必须引入隐藏层

多层感知机 (Multi-Layer Perceptron)：输入层 + 隐藏层 + 输出层的层次结构

12.2 多层前馈神经网络

结构特征：每层神经元与下一层全互连，同层无连接，无跨层连接。

12.3 万能近似定理

定理：只要隐藏层神经元数量足够多，仅需一个隐藏层的神经网络就能以任意精度近似任何连续函数。

条件说明
隐藏层数量仅需 1 层（但宽度需足够）
激活函数必须非多项式（线性激活函数再多层也只能表示线性函数）
定义域紧集 K（函数必须在有限范围内逼近）
实践限制：单层网络理论上可拟合一切，但所需神经元数可能随维度呈指数增长（维数灾难）。深层网络在表达层次结构数据时比单层网络高效得多。
