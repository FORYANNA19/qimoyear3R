第12讲：支持向量机 (Support Vector Machine, 支持向量机) 细读

Step 1 支持向量机 直观理解：最大化间隔
核心思想：寻找一个超平面，使得两类样本中离该平面最近的点（支持向量）之间的距离最大化

最大间隔 (Maximum Margin)

支持向量机 追求的不是简单地把两类分开，而是在正确分类的前提下，让分类边界距离最近的样本点尽可能远。这样得到的分类器具有最好的泛化能力。

支持向量 (Support Vectors)

那些正好落在间隔边界上的样本点，也就是距离超平面最近的点。支持向量机 的解只由这些支持向量决定，与其它样本点无关。支持向量的数量通常很少，因此 支持向量机 由少量"重要"样本决定。

支持向量机 的本质

求超平面使得离超平面最近的点之间的间隔最大化。最后决定超平面的只有支持向量，移动或去掉非支持向量不影响解。

三种情况：线性可分 → 硬间隔；广义线性可分 → 软间隔（引入松弛变量）；线性不可分 → 核技巧映射到高维空间。

几何间隔是实例点到超平面"带符号"的距离。当样本被正确分类时，几何间隔 = 点到超平面的距离。最大化几何间隔 = 正确分类 + 距离最大化。

Step 2 线性可分 支持向量机（硬间隔）
数据完全线性可分时的 支持向量机 原始问题与对偶问题

原始问题 (Primal Problem) 细读

优化目标

在正确分类所有样本的约束下，最小化权重向量的 L2 范数（等价于最大化间隔）。
$$\min_{w,b} \frac{1}{2}\|w\|^2$$ $$\text{s.t. } y_i(w \cdot x_i + b) \ge 1, \quad i=1,2,\dots,N$$
约束条件 \(y_i(w\cdot x_i + b) \ge 1\) 确保所有样本被正确分类且函数间隔至少为 1。目标函数 \(\frac{1}{2}\|w\|^2\) 等价于最大化间隔 \(\frac{2}{\|w\|}\)。

对偶问题 (Dual Problem) 细读

Lagrange 对偶

通过引入 Lagrange 乘子 \(\alpha_i \ge 0\)，将原约束优化问题转化为关于 \(\alpha\) 的对偶问题。对偶问题只涉及样本间的内积运算，为核技巧埋下伏笔。
$$\min_{\alpha} \frac{1}{2}\sum_{i=1}^{N}\sum_{j=1}^{N}\alpha_i\alpha_j y_i y_j (x_i \cdot x_j) - \sum_{i=1}^{N}\alpha_i$$ $$\text{s.t. } \sum_{i=1}^{N}\alpha_i y_i = 0,\quad \alpha_i \ge 0,\ i=1,2,\dots,N$$

KKT条件 条件

原始问题和对偶问题强对偶的充要条件：平稳性条件、原始可行性、对偶可行性、互补松弛性条件。Slater 条件成立时，凸优化问题的强对偶成立 (p*=d*)。
$$\text{互补松弛: } \alpha_i\big[y_i(w\cdot x_i+b)-1\big] = 0$$
若 \(\alpha_i > 0\)，则 \(y_i(w\cdot x_i+b)=1\)，该样本点在间隔边界上，即为支持向量。若 \(\alpha_i = 0\)，则该点对超平面无贡献。

解的形式

最终得到的权重向量 w 是支持向量的线性组合：
$$w^* = \sum_{i=1}^{N}\alpha_i^* y_i x_i$$ $$b^* = y_j - \sum_{i=1}^{N}\alpha_i^* y_i (x_i \cdot x_j),\quad \text{选 } j \text{ 使 } \alpha_j^* > 0$$
弱对偶只是 d* ≤ p*，强对偶才保证 d* = p*。Slater 条件（存在严格可行内点）是强对偶成立的常用充分条件。

Step 3 广义线性可分 支持向量机（软间隔）
数据大体线性可分但存在异常值时的处理方法

松弛变量 (Slack Variables) \(\xi_i\)

对每个样本引入 \(\xi_i \ge 0\)，允许其不满足函数间隔 ≥ 1 的约束。\(\xi_i = 0\) 表示完全满足硬间隔；\(\xi_i > 0\) 表示该样本有一定程度的"违规"。几何上，每个样本有自己的分割边界。

惩罚参数 C

C 是调节参数，在"间隔最大化"和"误分类惩罚"之间做权衡。C 越大，对误分类的惩罚越重，模型越倾向于减少错误但间隔变窄→可能过拟合；C 越小，模型越容忍错误→间隔更宽→可能欠拟合。

软间隔原始问题 细读

$$\min_{w,b,\xi} \frac{1}{2}\|w\|^2 + C\sum_{i=1}^{N}\xi_i$$ $$\text{s.t. } y_i(w\cdot x_i + b) \ge 1 - \xi_i,\ \xi_i \ge 0,\ i=1,\dots,N$$

软间隔对偶问题 细读

$$\min_{\alpha} \frac{1}{2}\sum_{i=1}^{N}\sum_{j=1}^{N}\alpha_i\alpha_j y_i y_j (x_i \cdot x_j) - \sum_{i=1}^{N}\alpha_i$$ $$\text{s.t. } \sum_{i=1}^{N}\alpha_i y_i = 0,\quad 0 \le \alpha_i \le C,\ i=1,\dots,N$$
注意：软间隔对偶问题与硬间隔的唯一区别是 \(\alpha_i\) 有上界 C，即 \(0 \le \alpha_i \le C\)。

软间隔支持向量分类

软间隔下，\(\alpha_i > 0\) 的实例点为支持向量，分三种情况：
条件\(\xi_i\)位置
\(\alpha_i < C\)\(\xi_i = 0\)在间隔边界上（正确分类，充分置信）
\(\alpha_i = C, 0 < \xi_i < 1\)\(0 < \xi_i < 1\)在间隔边界与超平面之间（正确分类但不充分）
\(\alpha_i = C, \xi_i > 1\)\(\xi_i > 1\)在误分一侧（分类错误）
\(\alpha_i = C, \xi_i = 1\)\(\xi_i = 1\)正好在超平面上
实际数据通常线性不可分，因此软间隔 支持向量机 应用更广。硬间隔 支持向量机 是软间隔在 C→∞ 时的特例。

Step 4 合页损失函数 (Hinge Loss)
软间隔 支持向量机 的另一种等价解释

合页损失函数

支持向量机 可以等价地表示为最小化正则化的合页损失：
$$\min_{w,b} \sum_{i=1}^{N} \big[1 - y_i(w\cdot x_i + b)\big]_+ + \lambda\|w\|^2$$
其中 \([z]_+ = \max(0, z)\) 表示取正部。第一项为 Hinge Loss，第二项为正则化项。

三种情况分析

① \(y_i(w\cdot x_i+b) \ge 1\)：正确分类且距离 ≥ 1，损失 = 0
② \(0 \le y_i(w\cdot x_i+b) < 1\)：正确分类但距离不足，损失在 (0,1] 之间
③ \(y_i(w\cdot x_i+b) < 0\)：分类错误，损失 > 1

代理损失函数 (Surrogate Loss)

0-1 损失函数非连续且非凸，直接优化困难。Hinge Loss 是凸函数且为 0-1 损失的上界，数学性质好，可作为代理损失函数。
损失函数表达式特性
Hinge Loss\([1 - y f(x)]_+\)凸函数，0-1 损失上界
Squared Loss\((1 - y f(x))^2\)凸函数
Squared Hinge Loss\(([1 - y f(x)]_+)^2\)凸函数，可微

Step 5 核技巧 (Kernel Trick) & 非线性 支持向量机
将线性不可分问题通过映射转为高维线性可分问题

核技巧基本思想

通过非线性变换 \(\phi: x \to \phi(x)\) 将输入空间映射到高维特征空间（希尔伯特空间），在新空间中样本变得线性可分，然后在新空间中训练线性 支持向量机。由于对偶问题只涉及内积 \(x_i \cdot x_j\)，直接将内积替换为核函数 \(K(x_i, x_j) = \phi(x_i)\cdot\phi(x_j)\) 即可，无需显式计算 \(\phi(x)\)。

核函数定义

设 X 为输入空间，H 为特征空间（希尔伯特空间），若存在映射 \(\phi: X \to H\) 使得对任意 \(x, z \in X\)，有 \(K(x, z) = \phi(x)\cdot\phi(z)\)，则称 \(K(x,z)\) 为核函数，\(\phi\) 为映射函数。

非线性 支持向量机 对偶问题 细读

$$\min_{\alpha} \frac{1}{2}\sum_{i=1}^{N}\sum_{j=1}^{N}\alpha_i\alpha_j y_i y_j K(x_i, x_j) - \sum_{i=1}^{N}\alpha_i$$ $$\text{s.t. } \sum_{i=1}^{N}\alpha_i y_i = 0,\quad 0 \le \alpha_i \le C$$
$$f(x) = \text{sign}\left(\sum_{i=1}^{N}\alpha_i^* y_i K(x_i, x) + b^*\right)$$

常用核函数 细读

核函数表达式特点
多项式核\(K(x,z) = (x\cdot z + 1)^p\)p 次多项式特征空间
高斯核 (径向基函数 (RBF))\(K(x,z) = \exp\left(-\frac{\|x-z\|^2}{2\sigma^2}\right)\)映射到无穷维空间，最常用

正定核的条件

对称函数 \(K(x,z)\) 为正定核的充要条件：对任意 \(x_1,\dots,x_m \in X\)，对应的 Gram 矩阵 \([K(x_i, x_j)]_{m\times m}\) 是半正定矩阵。
核函数在低维空间进行计算，却能等价于高维空间的内积效果——这是核技巧的核心优势，避免了显式高维映射的计算灾难。

将二分类 支持向量机 拓展到多分类和回归任务

多分类策略 细读

策略方法分类器数量
一对一 (一对一 (OvO)) 每两类之间训练一个 支持向量机，新样本由所有分类器投票决定 \(\binom{K}{2} = \frac{K(K-1)}{2}\)

支持向量回归 (支持向量回归 (SVR)) 细读

ε-不敏感管 (ε-insensitive tube)

支持向量回归 (SVR) 希望找到一个函数 \(f(x)\)，使得大多数训练点的预测偏差不超过 ε。落在 ε-管内的点不计损失，落在管外的点才受惩罚。这类似于 支持向量机 中落在间隔带内部的点才有损失。

支持向量机 学习算法统一框架 细读

  1. 选核函数 \(K(x,z)\) 和参数 \(C\)
  2. 构造并求解对偶问题：\(\min_\alpha \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j K(x_i,x_j) - \sum\alpha_i\)，s.t. \(\sum\alpha_i y_i=0,\ 0\le\alpha_i\le C\)
  3. 计算 \(w^* = \sum\alpha_i^* y_i x_i\)（线性情况）
  4. 选 \(\alpha_j^* \in (0,C)\)，计算 \(b^* = y_j - \sum\alpha_i^* y_i K(x_i, x_j)\)
  5. 决策函数：\(f(x) = \text{sign}(\sum\alpha_i^* y_i K(x_i, x) + b^*)\)
考研/期末必记公式：
① 原始问题：\(\min \frac{1}{2}\|w\|^2 + C\sum\xi_i\)
② 对偶问题：\(\min_\alpha \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j (x_i\cdot x_j) - \sum\alpha_i\)
③ 决策函数：\(f(x) = \sum\alpha_i^* y_i (x_i\cdot x) + b^*\)
④ KKT条件 互补松弛：\(\alpha_i[y_i(w\cdot x_i+b)-1+\xi_i] = 0\)
