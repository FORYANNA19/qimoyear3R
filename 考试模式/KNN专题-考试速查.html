KNN专题 — 考试速查

1. 三种评估方法速查 必考

方法训练集测试集核心特征$K=1$ 误差
回代法
(Resubstitution)		 全量全量 严重低估，仅作下界参考 0%（完美记忆）
留出法
(Holdout)		 训练集测试集 方差大，依赖划分 高（~40-60%）
留一法
(留一交叉验证 (LOOCV))		 n-1个剩余1个 偏差最小，计算开销大 高（~40-50%）
考点：回代法 $K=1$ 时误判率必为 0%（最近邻是自身）。留一交叉验证 (LOOCV) 的误差估计最接近真实泛化误差。

2. 错判率公式 必考

分类错判率： $$\text{Error Rate} = \frac{\text{误判样本数}}{\text{总样本数}} \times 100\% = \frac{1}{n}\sum_{i=1}^{n} \mathbb{I}(\hat{y}_i \neq y_i) \times 100\%$$

回归MSE： $$\text{均方误差 (MSE)} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

3. K值效应速记 必考

  • $K$ 太小：方差大，过拟合，决策边界复杂，对噪声敏感
  • $K$ 太大：偏差大，欠拟合，决策边界过度平滑
  • 最优 $K$：通过交叉验证（如LOOCV）选择使错判率最低的值

# 分类
fit <- knn(train = train_x, test = test_x, cl = train_y, k = 5)
err <- mean(fit != test_y) * 100

# 留一交叉验证 (LOOCV)
fit <- knn.cv(train = data_x, cl = data_y, k = 5)

# 回归 (caret::knnreg)
model <- knnreg(y ~ x1 + x2, data = train, k = 5)

mse <- mean((test$y - pred)^2)

# caret + kknn 调参
train(Purchase ~ ., data, method = "kknn",

      preProcess = c("center", "scale"))

5. 案例分析要点 高频

  • 预处理：数值变量标准化（center+scale），分类变量转因子（as.factor）
  • 调优参数：kmax（邻居数）、kernel（核函数）、distance（距离阶数）
  • 核函数对比：triangular > gaussian > rectangular（通常）
  • 变量重要性：varImp() 基于ROC曲线面积，KNN中仅供参考
考试陷阱：KNN回归不能用 class::knn（会强转为无意义因子），必须用 caret::knnreg 或 FNN::knn.reg。
