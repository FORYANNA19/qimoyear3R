第1讲-考试速查

课程简介与机器学习概论 · 核心公式 & 考点

一、必背定义 必考

1.1 Mitchell 机器学习定义

三要素：T（Task 任务）、P（Performance Measure 性能度量）、E（Experience 经验）

英文：A computer program is said to learn from experience E with respect to task T and performance measure P, if its performance at T improves with E.

问"P是什么"，答"预测天气"是错误的。P必须是可度量的性能指标，如"预测正确的概率"。

1.2 监督学习 vs 无监督学习

监督学习无监督学习
训练数据$$\{(\mathbf{x}_i, y_i)\}_{i=1}^{N}$$$$\{\mathbf{x}_i\}_{i=1}^{N}$$
特征有输入X + 输出标签Y只有输入X，没有Y
目标学习 X→Y 的映射描述数据的组织/聚集方式
典型任务分类、回归聚类、降维

1.3 分类 vs 回归

分类：输出Y是离散类别（垃圾/正常邮件、数字0-9）

回归：输出Y是连续数值（股价、交易量、温度）

二、核心公式速记 必考

2.1 损失函数一览

名称公式
0-1 损失$$L(Y, f(X)) = \mathbf{1}\{Y \neq f(X)\}$$
平方损失$$L(Y, f(X)) = (Y - f(X))^2$$
绝对值损失$$L(Y, f(X)) = |Y - f(X)|$$
对数损失$$L(Y, P(Y \mid X)) = -\log P(Y \mid X)$$

2.2 经验风险 vs 期望风险 vs 结构风险

期望风险（理论）： $$R_{\text{exp}}(f) = \mathbb{E}[L(Y, f(X))]$$ 经验风险（实际）： $$R_{\text{emp}}(f) = \frac{1}{N} \sum_{i=1}^{N} L(y_i, f(\mathbf{x}_i))$$ 结构风险（防过拟合）： $$R_{\text{str}}(f) = R_{\text{emp}}(f) + \lambda \cdot J(f)$$
期望损失依赖P(X,Y)不可知，实际用经验损失近似。样本量小时经验风险最小化容易过拟合，需用结构风险加正则化项。

2.3 线性回归（三要素）

要素内容
模型$$f(\mathbf{x}) = \mathbf{w}^T \mathbf{x} + b = \boldsymbol{\beta}^T \mathbf{x}$$
策略$$\min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2$$ （最小二乘 / L2损失）
算法解析解：$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^T \mathbf{X})^{-1} \mathbf{X}^T \mathbf{y}$$（正规方程）
迭代：梯度下降

2.4 梯度下降法

迭代公式： $$\boldsymbol{\beta}^{(k+1)} = \boldsymbol{\beta}^{(k)} - \eta \cdot \nabla J(\boldsymbol{\beta}^{(k)})$$ 其中 $$\eta$$ 是学习率（步长）。

2.5 岭回归（Ridge Regression）

$$\hat{\boldsymbol{\beta}}_{\text{ridge}} = (\mathbf{X}^T \mathbf{X} + \lambda \mathbf{I})^{-1} \mathbf{X}^T \mathbf{y}$$ 等价于：$$\min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2 + \lambda \|\boldsymbol{\beta}\|_2^2$$

2.6 逻辑回归

Sigmoid激活函数：$$\sigma(z) = \frac{1}{1 + e^{-z}}$$ 模型：$$P(Y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^T \mathbf{x} + b)$$ 对数几率：$$\log\frac{p}{1-p} = \mathbf{w}^T \mathbf{x} + b$$ 负对数似然：$$\ell(\mathbf{w}) = -\sum_{i=1}^{N} [y_i \log p_i + (1-y_i)\log(1-p_i)]$$ 梯度：$$\nabla \ell(\mathbf{w}) = -\mathbf{X}^T (\mathbf{y} - \mathbf{p})$$ 分类规则：$$\hat{y}=1 \iff P(Y=1|\mathbf{x}) \ge 0.5 \iff \mathbf{w}^T\mathbf{x}+b \ge 0$$
逻辑回归是线性分类器！决策边界是超平面 $$\mathbf{w}^T \mathbf{x} + b = 0$$，不要被Sigmoid函数迷惑。

四、考试易错陷阱 高频

1. 将"预测任务本身"误作P。要求给出P时应该回答性能度量指标（如正确率、误差率），而非任务描述。

2. 分类 vs 回归的误判：输出是离散类别 → 分类；输出是连续数值 → 回归（与输入变量类型无关）。

3. 混淆监督学习和无监督学习的数据格式：有无标签Y是唯一判断标准。

4. 经验风险最小化在小样本下容易过拟合，解决方法是结构风险最小化（加正则化项）。

5. 以为逻辑回归是非线性分类器。实际上它是线性分类器，Sigmoid仅用于概率映射。

五、考前必背清单

  1. ML = 数据 + 模型 + 策略 + 算法
  2. Mitchell定义的三要素：T、P、E
  3. 经验风险公式：$$R_{\text{emp}}(f) = \frac{1}{N}\sum L(y_i, f(x_i))$$
  4. 结构风险 = 经验风险 + λ·J(f)
  5. 线性回归解析解：$$\hat{\beta} = (X^TX)^{-1}X^Ty$$
  6. 梯度下降迭代式：$$\beta^{(k+1)} = \beta^{(k)} - \eta \nabla J$$
  7. Sigmoid函数：$$\sigma(z) = 1/(1+e^{-z})$$
  8. 逻辑回归负对数似然：$$-\sum[y_i\log p_i + (1-y_i)\log(1-p_i)]$$
  9. 逻辑回归 = 线性分类器（决策边界：$$w^Tx+b=0$$）
  10. No Free Lunch Theorem：没有一致最优算法
