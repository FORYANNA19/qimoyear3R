第11讲 考试速查：BP反向传播算法

上海对外经贸大学 · 数据挖掘与机器学习  |  2026年5月

一、多层感知机 (MLP) 网络结构速查

概念要点
多层前馈网络层间全互连；无同层/跨层连接
输入层 d 个神经元对应数据属性的维度
隐层 q 个神经元超参数，可调
输出层 l 个神经元对应目标变量维度
可训练权重数（不计偏置）d × q + q × l
激活函数 f(x)通常 Sigmoid激活函数：\(f'(x)=f(x)(1-f(x))\)
多层感知机 (MLP) vs 单层感知机隐藏层 → 能处理线性不可分问题
全用线性激活函数等价于简单的线性模型

二、核心符号速查表

符号含义
\(v_{ih}\)输入层第 i 个神经元 → 隐层第 h 个神经元 的连接权
\(w_{hj}\)隐层第 h 个神经元 → 输出层第 j 个神经元 的连接权
\(\gamma_h\)隐层第 h 个神经元的阈值 (bias)
\(\theta_j\)输出层第 j 个神经元的阈值 (bias)
\(\alpha_h = \sum_i v_{ih} x_i\)隐层第 h 个神经元接收到的输入
\(\beta_j = \sum_h w_{hj} b_h\)输出层第 j 个神经元接收到的输入
\(b_h = f(\alpha_h - \gamma_h)\)隐层第 h 个神经元的输出
\(\hat{y}_j = f(\beta_j - \theta_j)\)输出层第 j 个神经元的预测值

三、反向传播 算法完整步骤

Phase 1 — 前向传播 (Forward)

  1. 输入 \(\mathbf{x}_k\) → 计算 \(\alpha_h = \sum_i v_{ih} x_i\)
  2. 计算 \(b_h = f(\alpha_h - \gamma_h)\)
  3. 计算 \(\beta_j = \sum_h w_{hj} b_h\)
  4. 计算 \(\hat{y}_j = f(\beta_j - \theta_j)\)
  5. 计算损失 \(E_k = \frac{1}{2}\sum_j (\hat{y}_j - y_j)^2\)

Phase 2 — 反向传播 (Backward)

  1. 计算输出层误差：\(g_j = \hat{y}_j(1-\hat{y}_j)(y_j - \hat{y}_j)\)
  2. 更新输出层权重和阈值：\(\Delta w_{hj} = \eta g_j b_h\);  \(\Delta\theta_j = -\eta g_j\)
  3. 计算隐层误差：\(e_h = b_h(1-b_h)\sum_j w_{hj} g_j\)
  4. 更新隐层权重和阈值：\(\Delta v_{ih} = \eta e_h x_i\);  \(\Delta\gamma_h = -\eta e_h\)

四、必背公式卡

参数梯度更新量 \(\Delta\)
\(w_{hj}\) \(\frac{\partial E_k}{\partial w_{hj}} = -g_j b_h\) \(\Delta w_{hj} = \eta g_j b_h\)
\(\theta_j\) \(\frac{\partial E_k}{\partial \theta_j} = g_j\) \(\Delta\theta_j = -\eta g_j\)
\(v_{ih}\) \(\frac{\partial E_k}{\partial v_{ih}} = -e_h x_i\) \(\Delta v_{ih} = \eta e_h x_i\)
\(\gamma_h\) \(\frac{\partial E_k}{\partial \gamma_h} = e_h\) \(\Delta\gamma_h = -\eta e_h\)
输出层误差信号： \(g_j = \hat{y}_j(1-\hat{y}_j)(y_j - \hat{y}_j)\)
隐层误差信号： \(e_h = b_h(1-b_h)\sum_{j=1}^{l} w_{hj}\, g_j\)

五、链式法则推导 \(w_{hj}\) 梯度

$$\frac{\partial E_k}{\partial w_{hj}} = \underbrace{\frac{\partial E_k}{\partial \hat{y}_j}}_{\hat{y}_j - y_j} \cdot \underbrace{\frac{\partial \hat{y}_j}{\partial \beta_j}}_{f'(\beta_j - \theta_j)} \cdot \underbrace{\frac{\partial \beta_j}{\partial w_{hj}}}_{b_h}$$

\(w_{hj}\) → \(\beta_j\) → \(\hat{y}_j\) → \(E_k\)：三个环节，逐层链式相乘。核心数学工具是微积分中的链式法则。

六、Mini-Batch 速记

类型每步样本数特点
Full Batch全部 m 个梯度准但慢，内存大
随机梯度下降 (SGD)1 个快但方差大，"横冲直撞"
Mini-Batch32/64/128折中方案：无偏估计 + GPU 并行 + 噪声正则化

七、考试易错提醒

反向传播阶段计算什么？ 答案是"损失函数对网络中各个权值和偏置的梯度"，不是激活值，不是预测准确率。
"Batch"指的是什么？ 在更新一次参数之前，计算多个样本的平均梯度。不是指网络层数。
\(\eta\) 太大/太小的后果： 太大 → 振荡；太小 → 收敛慢。
线性激活 多层感知机 (MLP) = 线性模型： 多层线性变换复合仍是线性变换，等价于单层线性模型。
