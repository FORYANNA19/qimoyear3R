1. CART vs ID3/C4.5 必考

ID3 / C4.5CART
任务仅分类分类 + 回归
树结构多叉树二叉树（每次二分）
分裂准则(分类)信息增益 / 增益比Gini Index（最小化）
分裂准则(回归)—最小平方残差 MSPE

2. CART 算法三步骤

  1. 特征选择：分类用 Gini Index，回归用最小平方误差
  2. 决策树生成：基于训练数据，尽量生成大树
  3. 决策树剪枝：用验证集剪枝并选最优子树

3. 基尼指数公式 必考

概率分布基尼指数（K类）

Gini(p) = Σ pk(1-pk) = 1 - Σ pk²

二分类基尼指数

Gini(p) = 2p(1-p)

p=0.5时最大 → 0.5；p=0或1时最小 → 0

条件基尼指数（特征A，切分点a）

Gini(D, A=a) = (|D₁|/|D|)·Gini(D₁) + (|D₂|/|D|)·Gini(D₂)

4. 分类树 CART 算法流程

  1. 对当前结点每个特征A，对其每个取值a，按 A=a 将 D 分 D₁,D₂，计算 Gini(D, A=a)
  2. 选 Gini 最小 的特征和切分点，将数据分入两个子结点
  3. 对子结点递归直至停止

停止条件：①样本数<阈值 ②Gini<阈值 ③无更多特征

5. 回归树公式

回归树表达

f(x) = Σ cm·I(x ∈ Rm)

最优预测值（MSE损失下）

ĉm = (1/|Rm|)·Σ yi   (xi∈Rm)

= 该区域内所有 y 的 算术平均值（绝对值损失下为中位数）

分裂准则

minj,s [ ΣR₁(y-ĉ₁)² + ΣR₂(y-ĉ₂)² ]

固定j求最优s → 扫描所有j → 得最优(j, s)

6. CART 剪枝 必考

损失函数

Cα(T) = C(T) + α·|T|

C(T)=预测误差(Gini/均方误差 (MSE))，|T|=叶结点数，α≥0调节参数

α=0 → 整体树最优；α→∞ → 单结点树最优

节点临界值

g(t) = [C(t) - C(Tt)] / [|Tt| - 1]

流程

  1. 从 T₀ 底端开始，每次剪掉 g(t) 最小的节点，得 T₁, T₂, …, Tn
  2. 在验证集上交叉验证测试所有子树
  3. 选验证集误差最小的子树

7. Bootstrap & Bagging

Bootstrap

有放回抽 m 次，始终不被抽到的概率：(1-1/m)m → 1/e ≈ 0.368

约 63.2% 训练，36.8% 袋外 (OOB)

Bagging = Bootstrap + Aggregating

B次Bootstrap → 训练B个学习器 → 分类投票 / 回归平均

方差：Var = ρσ² + (1-ρ)σ²/B

Bagging 降低方差，不降低偏差；学习器独立 → 可并行

8. 随机森林 (RF) 必考

核心：Bagging + 随机特征子集

  1. Bootstrap有放回采样 N 个样本
  2. 每个节点从 M 个特征中随机选 m ≪ M 个，再选最优属性分裂
  3. 重复 B 次生成森林
  4. 分类：多数表决；回归：取平均
参数分类默认回归默认
m (mtry)⌊√M⌋⌊M/3⌋
最小节点样本15

树不剪枝，完全生长！

m → M 时 RF 退化为 Bagging

RF缺点：可解释性降低
