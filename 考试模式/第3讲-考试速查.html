K近邻法 K近邻法 — 考试速查

一、核心定义

概念定义
K近邻法基本分类与回归方法（Cover & Hart, 1968），不具有显式学习过程，属于懒惰学习 (Lazy Learning)
分类根据K个最近邻的类别，通过多数表决决定
回归取K个最近邻输出值的平均值作为预测值
三要素K值选择 + 距离度量 + 分类决策规则

二、核心公式

2.1 距离度量

$$\text{Minkowski: } L_p(\mathbf{x}, \mathbf{y}) = \left( \sum_{i=1}^{n} |x_i - y_i|^p \right)^{\frac{1}{p}}$$
p名称公式
1曼哈顿 | Block$$L_1 = \sum_{i=1}^{n} |x_i - y_i|$$
2欧氏 | Euclidean$$L_2 = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2}$$
∞切比雪夫 | Chebyshev$$L_\infty = \max_i |x_i - y_i|$$

2.2 余弦距离

$$\text{Cosine}(\mathbf{x}, \mathbf{y}) = 1 - \frac{\sum_{i=1}^{n} x_i y_i}{\sqrt{\sum x_i^2}\sqrt{\sum y_i^2}}$$

2.3 数据预处理

极差法 (Min-Max)

$$x'_{ij} = \frac{x_{ij} - \min_j}{\max_j - \min_j}$$

标准分数法 (Z-Score)

$$x'_{ij} = \frac{x_{ij} - \bar{x}_i}{\sigma_{x_i}}$$

2.4 分类决策（多数表决）

$$y = \arg\max_{c_j} \sum_{x_i \in N_k(x)} I(y_i = c_j)$$

在0-1损失下，多数表决规则 = 经验风险最小化。

误分类率：$$\frac{1}{k} \sum_{x_i \in N_k(x)} I(y_i \neq c_j) = 1 - \frac{1}{k} \sum I(y_i = c_j)$$

2.5 加权KNN预测

$$y = \arg\max_{c_j} \sum_{x_i \in N_k(x_0)} w_i \cdot I(y_i = c_j)$$

其中 $$w_i = K(d_i)$$，核函数满足：d=0时取最大值，单调递减的非负函数。

2.6 偏差-方差分解

$$\text{Err}(x) = \text{bias}^2(x) + \text{var}(x) + \sigma_\varepsilon^2$$

三、K值选择

K值效果问题
K=1（太小）模型复杂，决策边界曲折过拟合 Overfitting
K适中平衡偏差与方差需交叉验证选取
K=N（太大）模型最简单，预测恒为多数类欠拟合 Underfitting

最佳实践：用交叉验证法 (K-fold CV) 选取最优K；二分类选奇数以避免平票。

四、关键结论与陷阱

陷阱说明
维度灾难高维空间中距离趋于相近，"近邻"概念失效。$$\text{边长} \propto r^{1/p}$$
不归一化量级差异大的特征会主导距离计算，必须预处理
偶数K二分类可能出现平票，建议使用奇数K
不平衡数据K较大时少数类召回率极低
时间复杂度暴力搜索：$$O(N \times D)$$

五、模型评估方法

方法描述适用
旁置法 Hold-out随机分训练60-70%+测试大样本
K折交叉验证分K个子集轮流测试，取均值通用
留一法 LOO每次留1个测试，重复N次取均值小样本，偏差低

六、特征选择与降维

降低维度提升KNN表现：

  • 取值离散程度较大的变量
  • 与输出变量相关性强的变量
  • 使预测误差减小较多的变量

方法：前向序列选择、PCA

七、偏差-方差诊断

高偏差 (欠拟合)高方差 (过拟合)
训练误差高低
测试误差高高 + 与训练误差差距大
解救方法增加模型复杂度增加数据 / 正则化 / 简化模型
