第6讲：Bagging与随机森林 — 考试速查

一、Bagging 核心速查

算法步骤（4步）

  1. Bootstrap抽样：从N个样本中有放回抽N个 → 约63.2%样本被选中
  2. 训练基学习器：在抽样子集上独立训练一棵决策树
  3. 重复B次：得到B棵不同的决策树
  4. 聚合：分类→多数投票；回归→简单平均
分类投票： H(x) = argmaxy Σb=1B I(hb(x) = y)
回归平均： H(x) = (1/B) Σb=1B hb(x)
关键点：Bagging是并行式集成，树之间无依赖；主要降低方差，偏差几乎不变。

二、随机森林 核心速查

算法步骤（5步）

  1. 对每棵树，Bootstrap抽样N个样本
  2. 在每个节点分裂时：从M个特征中随机选m个（m ≪ M）
  3. 从m个特征中选最优分裂特征（如基尼指数最小）
  4. 递归分裂直到停止，生成完整决策树（不剪枝）
  5. 重复B次 → B棵树投票/平均
参数推荐值说明
B（树数量）500~1000越大越好，但边际递减
m（mtry）分类：√M；回归：M/3m=M时退化为Bagging
RF = Bagging + 特征随机。双重随机性（样本+特征）使树间去相关，方差比Bagging更低。当mtry接近总特征数时 → 退化为传统Bagging。

三、偏差-方差分解 速查

泛化误差 = Bias² + Variance + σ²
误差含义高意味着
Bias²预测期望 vs 真实值差距欠拟合 → 增加模型复杂度
Variance预测值在不同训练集上的波动过拟合 → 集成/正则化/增数据
σ²数据固有噪声不可降低
模型偏差方差策略
单棵决策树（深）低高—
Bagging≈单树↓ 显著降并行 + 投票平均
随机森林略增↓ 进一步降+ 特征随机 → 去相关
AdaBoost / Boosting↓ 显著降略增串行纠错 → 降低偏差
核心结论：Bagging/RF 降方差（适合复杂基学习器）；Boosting 降偏差（将弱学习器变强）。

四、OOB袋外误差 速查

  • 定义：利用Bootstrap中未被选中的36.8%样本作为验证集
  • 分类：对每个样本i，仅用不包含i的树投票，计算误分率
  • 回归：对每个样本i，用袋外树预测均值，计算MSE
  • 用途：无需单独验证集的模型评估、调参、特征重要性

五、AdaBoost 核心速查（串行对比）

核心思想

  1. 错分样本权重大：每轮提高上一轮错分样本的权重
  2. 好分类器权重大：误差率低的弱分类器在投票时权重更大

关键公式

误差率：em = Σ wi · I(Gm(xi) ≠ yi)
分类器权重：αm = ½ ln((1-em)/em)
最终分类器：G(x) = sign(Σ αm·Gm(x))
emαm说明
0.50等同于随机，无贡献
<0.5>0误差越小，权重越大

数学本质

AdaBoost = 加法模型 + 指数损失 + 前向分步算法
