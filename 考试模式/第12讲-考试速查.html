1. 核心优化目标（三层次递进）

硬间隔（线性可分）— 原始问题：

Primal $$\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t. } y_i(w\cdot x_i + b) \ge 1$$

软间隔（广义线性可分）— 原始问题：

Primal $$\min_{w,b,\xi} \frac{1}{2}\|w\|^2 + C\sum_{i=1}^{N}\xi_i \quad \text{s.t. } y_i(w\cdot x_i + b) \ge 1 - \xi_i,\ \xi_i \ge 0$$

核化软间隔（线性不可分）— 对偶问题：

Dual $$\min_{\alpha} \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j K(x_i,x_j) - \sum_i\alpha_i$$ $$\text{s.t. } \sum_i \alpha_i y_i = 0,\quad 0 \le \alpha_i \le C$$

2. KKT条件 条件（强对偶的充要条件）

KKT条件 $$\begin{aligned} \nabla_w L &= w - \sum\alpha_i y_i x_i = 0 &&\text{(平稳性)}\\ \nabla_b L &= -\sum\alpha_i y_i = 0 &&\text{(平稳性)}\\ y_i(w\cdot x_i + b) &\ge 1 - \xi_i,\ \xi_i \ge 0 &&\text{(原始可行性)}\\ \alpha_i &\ge 0,\ \mu_i \ge 0 &&\text{(对偶可行性)}\\ \alpha_i\big[y_i(w\cdot x_i + b) - 1 + \xi_i\big] &= 0 &&\text{(互补松弛)}\\ \mu_i \xi_i &= 0 &&\text{(互补松弛)} \end{aligned}$$

关键推论：\(\alpha_i > 0\) → 该点为支持向量；\(\alpha_i = 0\) → 对模型无贡献。

3. 必背公式清单

间隔 (Margin)：

$$\text{margin} = \frac{2}{\|w\|}$$

几何间隔：

$$\gamma_i = y_i\left(\frac{w}{\|w\|}\cdot x_i + \frac{b}{\|w\|}\right)$$

\(w^*\) 表达式：

$$w^* = \sum_{i=1}^{N}\alpha_i^* y_i x_i$$

\(b^*\) 计算：

$$b^* = y_j - \sum_{i=1}^{N}\alpha_i^* y_i K(x_i, x_j)$$

决策函数：

$$f(x) = \text{sign}\left(\sum_{i=1}^{N}\alpha_i^* y_i K(x_i, x) + b^*\right)$$

Hinge Loss：

$$L_{\text{hinge}} = \big[1 - y f(x)\big]_+$$

4. 常用核函数速查

核函数公式 \(K(x,z)\)映射空间使用场景
线性核\(x \cdot z\)原空间高维稀疏特征（文本）
多项式核\((x\cdot z + c)^d\)有限维已知特征交互阶数
高斯核 (径向基函数 (RBF))\(\exp\!\left(-\frac{\|x-z\|^2}{2\sigma^2}\right)\)无穷维通用，最常用
Sigmoid激活函数 核\(\tanh(\kappa x\cdot z + \theta)\)—神经网络视角

正定核判定：Gram 矩阵 \(G_{ij}=K(x_i,x_j)\) 半正定 ⇔ \(K\) 为正定核。

5. 参数 C 的影响

C 取值效果偏差-方差
\(C \to \infty\)退化为硬间隔，不允许错误可能过拟合（低偏差、高方差）
\(C\) 很大容忍少量错误，间隔窄倾向过拟合
\(C\) 适中平衡间隔与错误较好泛化
\(C\) 很小允许较多错误，间隔宽倾向欠拟合（高偏差、低方差）

7. 支持向量回归 (支持向量回归 (SVR)) 要点

  • 核心思想：找一个函数 \(f(x)\)，使大多数样本的偏差 ≤ ε
  • ε-不敏感管：落在管内的点不计损失，管外的点才有损失
  • 类比 支持向量机：间隔带 → ε-管，最大化间隔 → 最小化管外误差

⚠ 考试易错点

  • 特征缩放对 支持向量机 非常必要——依赖距离度量的模型都受量纲影响
  • 核技巧是在低维计算，等价于高维内积效果，并非显式映射
  • 支持向量是 \( \alpha_i > 0 \) 的点，不是所有样本点
  • \( \alpha_i = 0 \) 的点对超平面 完全无贡献
  • 软间隔 支持向量机 中 \(b^*\) 不唯一，但 \(w^*\) 唯一
  • 弱对偶：\(d^* \le p^*\)；强对偶：\(d^* = p^*\)（凸问题 + Slater 条件）
