一、感知机模型 必考

定义：f(x) = sign(w·x + b)，x ∈ ℝⁿ, y ∈ {-1, +1}

分离超平面：w·x + b = 0
w 是法向量，b 是截距，sign 是符号函数

本质：线性分类模型，判别模型

收敛条件：数据集必须线性可分（Linearly Separable）

二、核心公式速查 必背

公式名称表达式
点 x₀ 到超平面距离|w·x₀ + b| / ‖w‖
误分类点距离−yᵢ(w·xᵢ + b) / ‖w‖
损失函数 L(w,b)−Σ yᵢ(w·xᵢ + b) , xᵢ∈M
∇_w L(w,b)−Σ yᵢ xᵢ , xᵢ∈M
∇_b L(w,b)−Σ yᵢ , xᵢ∈M
权重更新（随机梯度下降 (SGD)）w ← w + η yᵢ xᵢ
偏置更新（随机梯度下降 (SGD)）b ← b + η yᵢ

三、感知机 随机梯度下降 (SGD) 算法 必考

  1. 初始化：w₀ = 0（或任意初值），b₀ = 0；学习率 η ∈ (0, 1]
  2. 随机选取训练数据 (xᵢ, yᵢ)
  3. 判断误分类：若 yᵢ(w·xᵢ + b) ≤ 0：
    w ← w + η · yᵢ · xᵢ
    b ← b + η · yᵢ
  4. 重复步骤 2-3，直至训练集中没有误分类点。

📌 记忆口诀："有错就改，w 加 ηyx，b 加 ηy"

四、激活函数对比 高频

激活函数公式输出范围特点使用场景
符号函数 (Sign) (符号)±1{-1, +1}不连续不光滑感知机
Sigmoid激活函数1/(1+e⁻ᶻ)(0, 1)光滑，导数 max 0.25，深层梯度消失多层感知机 (MLP) 传统
ReLU激活函数max(0, z)[0, +∞)x>0 导数为 1，缓解梯度消失现代深度学习
Tanh激活函数tanh(z)(-1, 1)零中心输出RNN/LSTM
注意：感知机激活函数 = Sign 函数（不是 Sigmoid激活函数、ReLU激活函数、Tanh激活函数）。Sigmoid激活函数 是 多层感知机 (MLP) 传统激活函数，用于 反向传播 算法推导。

五、反向传播 (BP) 算法速查

两个阶段：

  • 前向传播：输入 → 隐层 → 输出，计算损失
  • 反向传播：损失 → 梯度（链式法则），更新权值
链式法则核心公式：
∂E_k / ∂w_hj = (∂E_k / ∂ŷⱼ) · (∂ŷⱼ / ∂βⱼ) · (∂βⱼ / ∂w_hj)

📌 反向传播 权值更新公式（Sigmoid 激活）：Δw_hj = η · gⱼ · b_h，其中 gⱼ = ŷ(1-ŷ)(y - ŷ)

六、反向传播 三种变体对比 必考对比

特性标准 反向传播 (随机梯度下降 (SGD))累积 反向传播 (Batch)Mini-batch BP
更新频率每样本每 Epoch每小批量
收敛路径剧烈震荡非常平滑略震荡方向明确
内存需求极低极高适中（可调）

🔑 Mini-batch 反向传播 是现代深度学习工业标准。Batch Size 通常取 32、64 或 128。

七、核心性质速记 常考

问题答案
感知机收敛的前提？数据集线性可分
损失函数最小化什么？误分类点到超平面的总距离
η 影响收敛性吗？不影响，仅影响迭代次数
单层感知机的局限？不能解决 XOR（线性不可分）
多层感知机 (MLP) 解决 XOR 的关键？引入隐藏层
反向传播 核心数学工具？链式法则 (Chain Rule)
梯度消失的原因？Sigmoid激活函数 导数 max 0.25，连乘后指数衰减
ReLU激活函数 的主要优点？x>0 导数为 1，缓解梯度消失
万能近似定理条件？1个隐藏层 + 足够神经元 + 非多项式激活函数
线性激活函数的后果？多层等价于单层线性模型

八、多层感知机 (MLP) 结构要点

概念说明
多层前馈网络同层无连接、跨层无连接、仅相邻层全互连
参数数量（输入 d、隐层 q、输出 l）(d+1+l)q + l 个参数
隐层权值 d×q，隐层阈值 q 个，输出权值 q×l，输出阈值 l 个
万能近似定理1个隐藏层 + 足够神经元 = 可近似任何连续函数
但深层网络更高效（维数灾难）
