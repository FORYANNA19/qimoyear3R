四、神经元模型 (M-P) 高频

$$y = f\left(\sum_{i=1}^{n} w_i x_i - \theta\right)$$

xᵢ：输入信号 | wᵢ：连接权重 | θ：阈值 | f：激活函数

感知机激活函数：符号函数 sign(·)（不是 Sigmoid激活函数、ReLU激活函数、Tanh激活函数）

五、感知机模型公式 必考

$$f(x) = \text{sign}(w \cdot x + b)$$

w：权值向量（超平面法向量） | b：偏置（超平面截距）

感知机类型：判别模型（不是概率模型、生成模型、回归模型）

分离超平面：w·x + b = 0，将特征空间分为 +1 和 -1 两类

六、感知机损失函数 必考

损失函数：误分类点到超平面 S 的总距离（不是误分类个数）

$$L(w, b) = -\sum_{x_i \in M} y_i (w \cdot x_i + b)$$

性质：L(w,b) ≥ 0，关于 w,b 连续可导

M 为误分类点集合。对误分类点：-yᵢ(w·xᵢ + b) > 0

七、感知机学习算法 — 随机梯度下降 (SGD) 更新

随机选取一个误分类点 (xᵢ, yᵢ)：

$$w \leftarrow w + \eta y_i x_i$$ $$b \leftarrow b + \eta y_i$$

η：学习率，0 < η ≤ 1

几何直观：让超平面向该误分类点旋转，试图将其包含在正确的一侧

算法流程：

  1. 初始化 w₀, b₀
  2. 选取 (xᵢ, yᵢ)
  3. 若 yᵢ(w·xᵢ + b) ≤ 0：w ← w + ηyᵢxᵢ, b ← b + ηyᵢ
  4. 重复直到无误分类点

八、感知机重要性质速记 高频

问题答案
收敛前提条件？数据集必须线性可分
解是否唯一？不唯一（初值/误分类点顺序不同则解不同）
η 是否影响收敛？不影响收敛性（仅影响迭代次数）
能否解决 XOR？不能 → 需要多层感知机 (MLP))
批量学习 vs 在线学习？批量：全部数据再更新；在线：逐样本更新
对偶形式优势？D>>N 时通过 Gram 矩阵加速训练

九、梯度下降法公式

$$x^{(k+1)} \leftarrow x^{(k)} - \lambda_k \nabla f(x^{(k)})$$

负梯度方向是使函数值下降最快的方向

一阶泰勒展开：f(x) ≈ f(x(k)) + gkT(x - x(k))，gk = ∇f(x(k))

凸函数 + GD → 全局最优解

十、多层感知机 (MLP)) 与万能近似定理

多层前馈网络特征：

  • 每层神经元与下一层全互连
  • 同层无连接，无跨层连接

万能近似定理：

仅需 1 个隐藏层（宽度足够）+ 非多项式激活函数 = 可近似任何连续函数
⚠ 但单层所需神经元数可能随维度指数增长（维数灾难）
⚠ 深层网络在表达层次结构数据时比单层网络高效得多

激活函数要求：必须非多项式（线性激活 → 无论多少层都只能表示线性函数）

十一、考试易错清单

感知机部分：
❌ 感知机是生成模型/概率模型 → ✅ 判别模型
❌ 损失函数是误分类个数 → ✅ 误分类点到超平面总距离
❌ 激活函数是 Sigmoid激活函数 → ✅ 符号函数 sign(·)
❌ 解唯一 → ✅ 不唯一（依赖于初值和误分类点选取顺序）
❌ η 必须衰减才能收敛 → ✅ η 大小不影响收敛性
❌ 感知机输出概率，逻辑回归不输出概率 → ✅ 反了！感知机不输出概率
