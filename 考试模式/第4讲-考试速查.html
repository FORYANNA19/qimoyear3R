第4讲 决策树入门 — 考试速查

一、核心公式

1. 熵 (Entropy)

$$H(D) = -\sum_{k=1}^{K} p_k \log_2 p_k, \quad p_k = \frac{|C_k|}{|D|}$$

熵越大 = 不确定性越大。同类别 → H=0；均匀分布 → H=log2K（最大）。

2. 条件熵 (Conditional Entropy)

$$H(D \mid A) = \sum_{i=1}^{n} \frac{|D_i|}{|D|} \cdot H(D_i)$$

按特征 A 的 n 个取值划分 D 为 D1...Dn，各子集加权求和。

3. 信息增益 (Information Gain) — ID3准则

$$g(D, A) = H(D) - H(D \mid A)$$

= 互信息。ID3 选择 g 最大 的特征划分。

4. 信息增益率 (Gain Ratio) — C4.5准则

$$g_R(D, A) = \frac{g(D, A)}{H_A(D)}$$
$$H_A(D) = -\sum_{i=1}^{n} \frac{|D_i|}{|D|} \log_2 \frac{|D_i|}{|D|}$$

HA(D) = 特征 A 的固有值 (Intrinsic Value)。取值越多 → HA(D) 越大 → 增益率被惩罚。

5. 剪枝损失函数

$$C_\alpha(T) = C(T) + \alpha |T|$$

C(T) = ∑ Nt·Ht = 拟合度；|T| = 叶节点数；α = 调节参数。α 大 → 树小。

三、关键概念速记

概念一句话解释
决策树本质if-then 规则集合
决策树模型类型判别模型
根节点 → 叶节点路径互斥且完备，每条路径覆盖且唯一
决策边界形状与坐标轴平行/垂直
ID3 算法策略贪心算法 (Greedy)
ID3 分裂准则信息增益 g(D,A) 最大
C4.5 分裂准则信息增益率 gR(D,A) 最大
ID3 最大弱点偏向取值多的特征（如编号、日期）
C4.5 如何修正分母用 HA(D) 惩罚取值多的特征
连续值处理二分法 (Bi-partition) 找最优划分点
C4.5 缺失值处理按已知特征权重的比例分配到子节点

四、ID3 算法三种停止条件

#条件标签确定
(1)节点样本全属于同一类别该类
(2)属性集为空多数类
(3)样本集为空父节点多数类

五、剪枝策略对比

预剪枝后剪枝
机制提前停止（限制深度/最小样本）生成完整树后自下而上剪
效率高低（开销大）
泛化可能欠拟合更优

六、易错陷阱

⚠ 信息增益 vs 增益率：ID3 用信息增益（越大越好），C4.5 用增益率（除以固有值 HA(D) 修正偏好）。
⚠ ID3 偏向取值多的特征：如"编号"使条件熵=0，信息增益=H(D) 最大，但无泛化能力。
⚠ 决策树是判别模型：学习 P(Y|X)，非生成模型。对特征空间做划分，学习条件概率分布。
⚠ 贪心 × 全局最优：ID3/C4.5 每步选最优属性（贪心），不保证全局最优树。
